## 🔍 防坑特别提示清单

### 🔴 剧毒风险（必须警惕）

1. **厨房砸墙前必须确认承重状态** — 让物业出具建筑图纸
2. **闭水试验必须亲自去楼下看** — 拍照录像留存证据
3. **水电实测实量误差≤10%** — 拒绝绕线
4. **管线走向图必须提供** — 后期钻孔不会打爆水管
5. **1.2m×0.6m 大板砖铺贴** — 人工费比普通砖高 20-30 元/㎡，别被低价忽悠

### 🟠 危险等级（需要重点关注）

1. **材料品牌型号必须具体化** — "多乐士致悦"而非"多乐士"
2. **拒绝"按实结算"** — 改为"增项≤5%"
3. **防水质保≥5 年** — 国家标准不能打折
4. **大板瓷砖采购** — 需提前定制，工期预留 15-20 天

### ⚠️ 谨慎提醒（注意核对）

1. **调色额外收费** — 深色漆加收¥20/㎡
2. **大型吊灯安装费** — ¥50-200/个另计
3. **垃圾清运费含不含物业费** — 有些小区另收¥200-500/车
4. **成品保护常被遗漏** — 特别关注电梯和入户门
