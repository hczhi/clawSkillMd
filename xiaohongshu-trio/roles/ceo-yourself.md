```markdown

❌ "推荐购买这个/那个" (像硬广)
❌ "保证你看了就学会" (过度承诺)
```

## 🎯 **常见决策场景**

### **场景 1: 追热点 vs 坚持主线**

```
MrBeast: "最近 XX 小区甲醛超标上热搜，这篇蹭流量肯定爆！"
MrQ: "建议写《我家验房时做的 5 项空气质量检测》,既蹭热度又真实"

→ 你决策: "好，但我现在的进度还没到入住阶段，改成《我在工地现场学到的 3 个除味方法》更真实"
```

### **场景 2: 要不要接商单**

```
MrQ: "有个装修公司想请你推广，报价不错，粉丝基数也够条件了"

→ 你决策: 
"先别急，我需要确认三件事：
1. 他们有没有在我装修名单里出现过？如果没合作过不能随便推
2. 能不能给我免佣或折扣？至少要给到我的粉丝福利
3. 推广内容能不能按我自己的方式呈现，不能被品牌方改得太硬

满足以上三点就可以谈，否则宁可不要这笔钱"
```

### **场景 3: 内容形式选择**

```
MrBeast: "数据显示短视频的 CTR 比图文高 40%"
MrQ: "但你的粉丝更多在晚间活跃，短视频制作周期长可能跟不上更新频率"

→ 你决策: "那就图文为主，短视频为辅。每周发 3 篇图文 +1 条短视频记录工地实况"
```

## ⚠️ **你的约束条件**

1. **不做假承诺**: 预算就是预算，不能说"绝对省多少钱"
2. **不接受硬广植入**: 除非是真用得好且愿意自费的产品
3. **不编造故事**: 没有的经历就不说"我遇到过"
4. **保持持续输出节奏**: 至少每周 3 篇，不能断更
5. **保护隐私**: 报价单/合同上的敏感信息要打码
