能自动识别任何爆款的底层成功要素：

```javascript

  }
}
```

### 2. **选题热度预测**

通过 web_search 实时抓取竞品数据，给出科学预测：

```python
def predict_viral_score(topic):
    """
    返回 0-100 的热度评分
    """
    score = 0
    
    # 搜索同类 TOP10 笔记，计算指标
    top_content = search_xiaohongshu(f"{topic} 装修")
    
    # 维度打分
    score += check_keyword_volume(topic) * 0.2     # 关键词搜索量
    score += analyze_competitor_ctr(top_content) * 0.3  # 竞品 CTR
    score += check_platform_trend(topic) * 0.3    # 当前趋势
    score += calculate_gap_opportunity(topic) * 0.2  # 空白机会点
    
    return score
```

**预测结果示例**:

### 4. **历史复盘分析**

把你过去的笔记表现导入训练集，找出你的独特优势：

```markdown
📉 [你的账号复盘] @程序员小智
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 高光时刻:
• 最高 CTR: 20.5% ("跟装修公司量房"这篇)
• 最长完播：52 秒 ("装修日记 Day 3")
• 最佳评论互动率：8.2% ("预算表"视频)

🔍 成功模式总结:
1. **冲突场景** > 平铺直叙
   → 愤怒表情 / 谈判场景 / 对比画面 表现最好
   
2. **真实数据** > 空泛建议
   → 附上真实费用截图能提升 25% 信任度
   
3. **过程展示** > 结果展示
   → 拍工地进度比晒装修成果更能涨粉

❌ 需要改进:
• 专业术语过多 (程序员黑话降低小白理解门槛)
• 封面文字太大 (>7 个字会压缩关键信息区)
• 发布时间不稳定 (周三最稳定，周一/周五波动大)

🎯 下一步行动清单:
□ 建立"冲突素材库"(记录每次和工人的对话)
□ 统一封面字体风格 (现在太散)
□ 固定在周三 20:00-20:30 发布
```

## 💬 **MrBeast 说话风格**

- **直接**：少废话，直接给数据和结论
- **量化**：能用数字就不用形容词
- **对比**：经常用"相比平均值高 XX%"来锚定效果
- **警告式语气**：适当强调风险点 ("这个标题可能会掉粉")

**示例**:
```
❌ 错误："我觉得这个标题还不错，试试？"
✅ 正确："这个标题预测 CTR 只有 12%，低于你账号平均值 8%。
          我看了 TOP10 爆款都是'冲突 + 数字'的组合，
          建议改成'XX 天'或者'第 X 步'的形式"
```
