## 🎯 **二、笔记推荐算法**

### **2.1 漏斗式分发机制**

```
第一步：标签匹配 → 200-500 曝光（精准粉丝 + 兴趣人群）
   ↓ [CTR 达标？] 
第二步：一级流量池 → 2k-5k 曝光（扩大兴趣圈层）
   ↓ [完读率>60%? 收藏率>5%?]
第三步：二级流量池 → 10k-50k 曝光（更广泛推荐）
   ↓ [评论深度足够？分享率高？]
第四步：热门池 → 10w+ 曝光（登上同城/发现页）
```

### **2.2 核心考核指标**

| 指标 | 及格线 | 优秀线 | 权重 |
|------|-------|-------|------|
| **CTR**(点击率) | 12% | 20%+ | 30% |
| **完读率** | 45% | 65%+ | 25% |
| **收藏率** | 2% | 5%+ | 20% |
| **评论率** | 1% | 3%+ | 15% |
| **分享率** | 0.5% | 1.5%+ | 10% |

**计算公式**:
```python
笔记得分 = CTR×0.30 + 完读率×0.25 + 收藏率×0.20 + 评论率×0.15 + 分享率×0.10

if 得分 > 0.25: 进入下一级流量池
elif 得分 < 0.15: 停止推荐
else: 保持在当前池观察
```

## ⚠️ **四、违规红线**

### **4.1 绝对禁止行为**

| 类型 | 具体表现 | 处罚等级 |
|------|---------|---------|
| **导流** | 微信号/二维码/二维码图片/手机号 | 封禁笔记 -7 天内不展示 |
| **广告法违禁词** | 第一/独家/最低价/百分百有效 | 限流至仅粉丝可见 |
| **诱导行为** | "点赞关注领资料"/"评论区回复 XX" | 取消该条笔记流量分发权 |
| **外链引流** | 淘宝链接/天猫同款/拼多多 | 直接屏蔽不推荐 |
| **虚假宣传** | "限时抢购"/"最后 XX 名"无证据 | 警告 → 短期降权 |

## 🎯 **六、爆款特征总结**

### **6.1 CTR 触发器**（封面 + 标题）

```markdown
✅ 高 CTR 元素组合:

【标题】
• 数字具体化："7 个""5 分钟""3 万块"
• 冲突叙事："差点被坑""谈了 X 天"
• 悬念设置："没人告诉你""秘密"
• 痛点直击："小白必看""后悔没早看"

【封面】
• 真人表情特写（愤怒/惊讶/无语）
• 真实场景实拍（工地/合同/账单）
• 对比视觉（左边❌右边✅）
• 大字报风格（3-5 个字 + 醒目色）
```

### **6.2 AVD 触发器**（完播率）

```markdown
✅ 高完播率内容结构:

【开头 30 秒钩子】
• 抛出争议性结论："装修公司最喜欢骗这三种人"
• 承诺明确价值："看完这篇能省 2 万块"
• 情绪共鸣点："我跟你一样，也是第一次装修"

【中间节奏把控】
• 每 3 段一个小惊喜（新信息/反转/案例）
• 避免纯理论输出（多配真实截图）
• 语言口语化（像跟朋友聊天）

【结尾引导】
• 提问留白："你家遇到过类似情况吗？"
• 收藏动机："这份清单请收好"
• 互动邀请："评论区告诉我你的预算"
```

```python
def compliance_check(content):
    """
    合规性审查
    """
    violations = []
    
    # 检测违禁词
    for word in risky_words.sensitive:
        if word in content.title:
            violations.append(f"标题包含敏感词：{word}")
    
    # 检测导流信息
    if re.search(r'微信|VX|weixin|qq|电话号码', content.body):
        violations.append("检测到导流信息")
    
    return violations

def seo_score(title, body, tags):
    """
    SEO 评分系统 (0-100)
    """
    score = 0
    
    # 关键词密度检查
    main_keyword_count = count_keyword(title, body, MAIN_KEYWORDS)
    score += min(main_keyword_count * 10, 40)
    
    # 标签相关性
    tag_relevance = calculate_relevance(tags, TOPIC)
    score += tag_relevance * 30
    
    # 内容长度合理性
    ideal_length = 300 - 800
    if ideal_length <= len(body) <= 1500:
        score += 20
    
    # 排版可读性
    has_emojis = count_emojis(content) >= 5
    has_section_headers = count_linebreaks(content) >= 3
    score += (has_emojis and has_section_headers) * 10
    
    return score
```
