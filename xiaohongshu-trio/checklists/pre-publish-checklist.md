# ✅ 发布前检查清单

> MrQ 的合规审查工具——确保内容安全、优质、符合人设


## 📋 必查项目 (10 项)

### 1️⃣ 【高优先级】标题 SEO 优化

```markdown
□ 标题前 5 个字包含核心搜索词？  
□ 标题总长度≤20 字（移动端显示完整）？  
□ 没有使用 emoji 占位（影响搜索抓取）？  

【检查方法】复制标题到搜索框，看能否被精准匹配
```

**✅ 正确示例**:
- "水电改造避坑指南！这 5 处最容易被坑" → "水电改造"+"避坑"都是关键词
- "卫生间防水做多高？这篇说清楚了" → 直接回答搜索意图

**❌ 错误示例**:
- "💡装修干货！千万别踩这些坑😱" → 没有明确搜索词
- "震惊！你绝对想不到..." → 纯标题党，无价值信息


---

### 2️⃣ 【高优先级】封面图冲击力

```markdown
□ 能在 0.3 秒内抓住注意力吗？（自测：快速扫一眼再移开，能记住什么？）
□ 大字标题≤3 个字，超粗体？
□ 数字/价格有圆圈或箭头标注？
□ 人物表情是否符合内容情绪？（惊讶>开心>认真）
□ 文字与背景对比度≥70%?

【自测方法】把图片缩小到手机尺寸再看一遍
```

**✅ 合格标准**:
- 左图：原始报价单扫描件 (打码)
- 右图：Excel 拆解表 (高亮异常项)
- 大字："贵了 3 万？"
- 人物：右下角露出惊讶表情

**❌ 不合格**:
- 纯文字海报（像 PPT 截图）
- 色调太接近看不清重点
- 人物太小或完全不出镜


---

### 3️⃣ 【高优先级】开头 Hook 设计

```markdown
□ 前三行是否抛出钩子？（不是"大家好今天来分享..."这种废话）
□ 是否用了损失厌恶原理？（避免失去 > 获得收益）
□ 是否有人物代入感？（用"我"而非"你"）

【检查方法】自己读一遍开头，判断会不会想继续往下看
```

**✅ 好开头**:
- "花了 20 万才发现，防水这 3 件事没人告诉我！"
- "本来以为 9 万预算够用了，拆开报价单才发现多花了 3 万"

**❌ 坏开头**:
- "大家好，我是程序员小智，今天来聊聊装修防水"
- "很多人问我卫生间防水怎么做，这篇文章告诉你答案"


---

### 4️⃣ 【高优先级】违禁词扫描

```markdown
□ 有没有使用极限词？(最/第一/顶级/绝对/百分百)
□ 有没有承诺性表述？( guaranteed/稳赚不赔/一定有效)
□ 有没有诱导性词汇？(赚钱/暴富/躺赚/速成)
□ 有没有医疗/健康宣称？(治愈/疗效/消炎/排毒)

【自查方法】把文案复制到在线违禁词检测工具
```

**常见雷区**:
- ❌ "最好的防水方案" → ✅ "我觉得不错的方案"
- ❌ "保证让你省 3 万" → ✅ "我实际省了 3 万"
- ❌ "百分百成功" → ✅ "成功率很高"
- ❌ "根治甲醛问题" → ✅ "帮助降低甲醛浓度"


---

### 5️⃣ 【中优先级】营销软广嫌疑

```markdown
□ 如果有推荐产品，是否真实用过？
□ 是否过度强调购买链接/微信号？
□ 是否有明显的商业推广痕迹但伪装成个人分享？

【判断标准】如果是朋友看到这个笔记，会觉得是广告还是真心推荐？
```

**✅ 安全做法**:
- "我自己用了三个月觉得不错，附淘宝链接"
- "某品牌的 XX 系列性价比很高（非广告，自费购买）"

**❌ 危险做法**:
- 连续三篇都推同一个品牌
- 评论区统一引导加私信买货
- 不标注合作但明显是商单


---

### 6️⃣ 【高优先级】隐私信息脱敏

```markdown
□ 报价单/合同上有无敏感信息？(价格细节/联系方式/地址)
□ 工地照片是否拍到门牌号/小区名？
□ 个人隐私是否已打码？(电话号码/姓名全名/身份证号)

【处理方法】用修图软件模糊处理或用马赛克遮挡
```

**必须打码的信息**:
- 身份证号码
- 房产证号
- 手机号/微信号
- 具体门牌地址
- 银行卡号

**可以保留的信息**:
- 总价金额（如"花了 8,200 元"）
- 商家名称（如"XX 装修公司"但不写具体门店）
- 城市级别（如"广州"但不是"天河区 XX 路"）


---

### 7️⃣ 【中优先级】评论区预埋互动

```markdown
□ 是否在文末设置了提问或引导语？
□ 是否准备好应对负面评论的话术？
□ 是否需要自己先留一条评论带节奏？

【预埋话术示例】
- "需要的姐妹扣 1，我发你自查表"
- "你们装修踩过最大的坑是什么？评论区告诉我"
- "有任何问题欢迎留言，看到必回"
```

---

### 8️⃣ 【低优先级】发布时间窗口

```markdown
□ 是否符合目标人群活跃时段？
   - 工作日：7-9 点 / 12-13 点 / 20-23 点
   - 周末：10-12 点 / 15-17 点 / 19-23 点
   - 黄金时间：周二/周四晚 20:00-22:00

□ 如果现在不是黄金时间，是否可以定时发布？
```

**建议发布时间**:
- 干货类：周三/周五晚 20:00-22:00
- 情感类：周一早上 8:00-9:00
- 热点类：事件发生后 2 小时内越快越好


---

### 9️⃣ 【中优先级】人设一致性校验

```markdown
□ 这篇内容符合"程序员小智"的人设吗？
□ 语气是否过于官方/说教/营销腔？
□ 是否有真实的故事和细节支撑？

【自检问题】
- 如果我朋友看到这个，会觉得是我写的吗？
- 这篇内容里，有多少是我的真实经历？
- 有没有为了数据而违背本心的地方？
```

**✅ 符合人设的表达**:
- "说实话，刚开始我也这么想..."
- "花了整整 3 个月才搞懂..."
- "有一说一，这个钱花得值"

**❌ 不符合人设的表达**:
- "你们一定要听我的"
- "听我说完再下结论"
- "保证你看了就学会"


---

### 🔟 【低优先级】长尾价值评估

```markdown
□ 这篇内容一个月后还有参考价值吗？
□ 是否容易被搜索到？(关键词布局是否充分)
□ 是否需要做 ALT 文本优化？

【优化建议】
- 正文自然插入 3-5 个长尾词
- 图片 ALT 描述也带上关键词
- 话题标签至少覆盖一级/二级/三级词
```
