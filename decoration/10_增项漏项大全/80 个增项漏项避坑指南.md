# 装修中 80 个增项漏项避坑指南

本清单汇总了装修过程中最常见的增项和漏项陷阱，按施工阶段分类，供报价审核时逐项核对使用。

---

## 🚧 拆除阶段（4 项）

### 序号 1：铲墙皮
| 项目 | 详情 |
|------|------|
| **常见套路** | 不要光听装修公司忽悠，要根据墙面腻子质量怎么样，是否一手摸掉白，再决定铲不铲 |
| **收费标准** | 一般在 8 元/㎡左右 |
| **铲到什么程度** | 铲到水泥层还是只铲腻子层？差价大！ |
| **防坑要点** | ✅ 小面积测试 - 先铲一角看看腻子附着力<br>✅ 明确价格包含范围 - 是否含垃圾清运 |

### 序号 2：垃圾清运
| 项目 | 详情 |
|------|------|
| **常见套路** | 铲除墙体、墙壁铲灰、拆改墙体等装修阶段产生的垃圾，是否会把垃圾打包到垃圾清运点，是否需要额外收费 |
| **计费方式** | 按车（300-500 元/车）或按吨（50-80 元/吨） |
| **防坑要点** | ✅ 确认"楼下搬运费"是否已含<br>✅ 问清物业指定清运点距离 |

### 序号 3：保温层拆除
| 项目 | 详情 |
|------|------|
| **常见套路** | 阳台如果贴砖，保温层就需要拆除，否则挂不住砖 |
| **防坑要点** | ✅ 提前问清楚是否必须拆除<br>✅ 有些情况下可以保留并直接贴瓷砖 |

### 序号 4：拆除后修补
| 项目 | 详情 |
|------|------|
| **常见套路** | 铲除过程会对地面造成一定的破坏，要进行修补 |
| **防坑要点** | ✅ 要求报价时就包含修复费用<br>✅ 避免拆完后再加价 |

## 🧱 新建墙体阶段（6 项）

### 序号 5：卫生间防水地梁
| 项目 | 详情 |
|------|------|
| **常见套路** | 卫生间调整布局的情况下，需新建防水地梁，能避免缝隙，防止外墙因为水汽潮湿而霉变脱落 |
| **防坑要点** | ✅ 必须做！这是国标要求<br>✅ 确认高度≥200mm |

### 序号 6：新建过门石梁
| 项目 | 详情 |
|------|------|
| **常见套路** | 新建或调整原门洞，需新建过头门梁才能保证门的稳定性 |
| **计费差异** | 间清楚不需要单独收费，过头梁是浇筑，还是买成品的 |
| **防坑要点** | ✅ 确认是否含在砌墙费用中<br>✅ 成品梁更便宜但强度不如浇筑 |

### 序号 7：新建墙体抹灰
| 项目 | 详情 |
|------|------|
| **常见套路** | 间清楚是单面抹灰还是双面抹灰，养护时间多久 |
| **防坑要点** | ✅ 双面抹灰才结实<br>✅ 养护不足后期会开裂 |

### 序号 8：新建墙体植入钢筋
| 项目 | 详情 |
|------|------|
| **常见套路** | 新建墙体要每间隔 50 公分植入钢筋，要打植筋胶 |
| **单价参考** | 植筋间距≤500mm |
| **防坑要点** | ✅ 必须执行！防止新旧墙体分离<br>✅ 验收时检查植入深度 |

### 序号 9：顶面斜砌
| 项目 | 详情 |
|------|------|
| **常见套路** | 新建墙体顶面要采用斜砌工艺，以免后期墙体沉降，导致顶面拼接处开裂 |
| **防坑要点** | ✅ 必须等待新墙沉降稳定后再斜砌<br>✅ 通常要求砌筑完成 7-14 天后进行 |

### 序号 10：新建墙体挂网
| 项目 | 详情 |
|------|------|
| **常见套路** | 新建墙体与旧墙体交界处需要挂钢丝网 |
| **规格要求** | 钢丝网宽度≥300mm，每边各 150mm |
| **防坑要点** | ✅ 一定要挂！防止裂缝<br>✅ 确认网格密度和材质 |

## ⚡ 水电阶段（15 项）

### 序号 11：水电计价
| 项目 | 详情 |
|------|------|
| **常见套路** | 传统的收费方式是按照米数或者点位收，现在已经很少用这两种方式了，因为不好计算价格 |
| **建议方案** | ✅ 建议预估好开关插座、灯具等的数量，谈一口价 |
| **防坑要点** | ❌ 拒绝纯米数计费 → 容易绕线虚报长度 |

### 序号 12：空调孔
| 项目 | 详情 |
|------|------|
| **常见套路** | 在水电定位的时候，要问清楚现有空调孔是否需要更改高度，要更改的话那就要重新打孔 |
| **收费陷阱** | 可以自己请外面的师傅打孔，会便宜很多，不然等装修公司水电施工的时候他们会以赶进度为由自己打孔，他们打孔普遍都是很贵的 |
| **防坑要点** | ✅ 提前测量现有孔径和位置<br>✅ 自找师傅约 50-80 元/个 vs 装修公司 150-300 元/个 |

### 序号 13：水电开槽
| 项目 | 详情 |
|------|------|
| **常见套路** | 1. 水电均分为明装暗装，价格不一样，可提前沟通明装还是暗装 |
| **防坑要点** | ✅ 暗装美观但贵且伤墙体<br>✅ 明装可检修但影响美观 |

### 序号 14：强弱电箱安装
| 项目 | 详情 |
|------|------|
| **常见套路** | 户型改造可能会造成强弱电箱移位，需要重新开槽安装，一般 300-500 元/个 |
| **防坑要点** | ✅ 确认是否包含在改造预算内<br>✅ 新箱体材料品牌要写明 |

### 序号 15：水电走顶加钱
| 项目 | 详情 |
|------|------|
| **常见套路** | 提前确定水电是走地还是走顶，大部分装修公司水电走地的，如果想走顶是另外计算价钱，那在定装修之前就得确认好，这样好谈价格 |
| **防坑要点** | ✅ 厨卫建议走顶 → 便于检修<br>✅ 客厅走地更省钱但维修麻烦 |

### 序号 16：包保温棉
| 项目 | 详情 |
|------|------|
| **常见套路** | 水管走顶要包保温棉，以防冷凝水 |
| **防坑要点** | ✅ 必须做！北方尤其重要<br>✅ 确认保温材料品牌和厚度 |

### 序号 17：水电材料
| 项目 | 详情 |
|------|------|
| **常见套路** | 签订合同的时候一定要注明好所使用的水电材料及型号，避免使用差的材料，后期要升级可是一大笔费用 |
| **防坑要点** | ✅ 水管：PPR 抗菌管（如伟星、日丰）<br>✅ 电线：国标阻燃线（如熊猫、德力西） |

### 序号 18：水电穿线
| 项目 | 详情 |
|------|------|
| **常见套路** | 一根电管内安装 3 根电线，如果水电是按照米数计费，警惕一根电管内只安装一根电线，故意增加米数来提高水电改造的费用 |
| **安全标准** | 一根管内电线截面积不超过管径的 40% |
| **防坑要点** | ✅ 提前约定最大穿线数量<br>✅ 避免后期以散热为由增加管线 |

### 序号 19：水电走顶
| 项目 | 详情 |
|------|------|
| **常见套路** | 水电一般走地，厨卫水电集中建议走顶，更好维修，但是价格会更贵，需提前确认清楚 |
| **单价对比** | 走顶比走地约贵 30-50% |

### 序号 20：电线平方
| 项目 | 详情 |
|------|------|
| **常见套路** | 一般来说，普通照明用 2.5 平方为主国可，没有必要全屋用 4 平方线。但是入户建议 10 平方线，部分大功率设备或电器需要 4 或 6 平方线。电线平方越大，价格越贵，也要提前确认清楚 |
| **标准配置** | 
- 照明：1.5-2.5㎡
- 普通插座：2.5㎡
- 空调/厨房：4㎡
- 入户主线：10-16㎡ |

### 序号 21：水电材料（品牌核验）
| 项目 | 详情 |
|------|------|
| **常见套路** | 水电材料是重中之重，提前确认好品牌、型号、规格等信息，确认是正规的大牌子，避免到装的时候发现用的是杂牌，被坐地起价 |
| **防坑要点** | ✅ 进场时当面验货，拍照留底<br>✅ 假货双倍赔偿写进合同 |

### 序号 22：强弱电交叉
| 项目 | 详情 |
|------|------|
| **常见套路** | 强弱电交叉的地方要裹锡箔纸，以防信号干扰 |
| **工艺标准** | 交叉处包裹≥300mm 长锡箔纸 |

### 序号 23：提前定好回路
| 项目 | 详情 |
|------|------|
| **常见套路** | 一般一个两居室 9 个回路就够了，冰箱记得单走一路 |
| **建议配置** | 
- 照明：1 回路
- 普通插座：2-3 回路
- 厨房：1 独立回路
- 卫生间：1 独立回路
- 空调：每台 1 回路 |

### 序号 24：零线
| 项目 | 详情 |
|------|------|
| **常见套路** | 如果做全屋只能需要走零线，提前沟通好价格 |
| **防坑要点** | ✅ 智能家居需要零线 → 提前预留<br>✅ 确认是否含在全屋布线中 |

### 序号 25：卫生间回填
| 项目 | 详情 |
|------|------|
| **常见套路** | 定装修的时候不告诉你是用什么进行回填，但实际上回填分为回填宝、陶粒渣渣回填，每种的价格都不一样，要问清楚，最好不要用建筑垃圾回填 |
| **推荐方案** | ✅ 陶粒回填最轻最环保（120-180 元/m³）<br>❌ 建筑垃圾回填便宜但重且易开裂 |

## 💧 防水阶段（7 项）

### 序号 26：防水次数
| 项目 | 详情 |
|------|------|
| **常见套路** | 卫生间墙面要涂刷 2-3 遍，墙面用刚性防水，地面用柔性防水，工艺提前确定清楚 |
| **防坑要点** | ✅ 十字交叉涂刷法（横刷 + 竖刷）<br>✅ 每遍间隔时间要够（表干后再刷下一遍） |

### 序号 27：防水高度
| 项目 | 详情 |
|------|------|
| **常见套路** | 可能报价时的高度说 1.2 米，后面告诉你做 1.2 有哪些弊端，推荐你加到 1.8 米；又要加钱了，都得提前确认好 |
| **国标要求** | ✅ 淋浴区：1.8m（建议做到顶）<br>✅ 干区：30cm<br>✅ 窗台：30cm |
| **防坑要点** | ✅ 报价时就确认高度，避免后期加价<br>✅ 门口要做止水墩，防止水从地下渗透 |

### 序号 28：防水材料
| 项目 | 详情 |
|------|------|
| **常见套路** | 辅料的所有品牌都要确定好，具体的规格型号，在开工前都要写明，不然后期用不好的，再换就要你升级加钱 |
| **推荐品牌** | 
- 刚性防水（墙面）：雨虹 100、德高 K11 通用型
- 柔性防水（地面）：雨虹 200、德高 K11 柔韧型 |

### 序号 29：闭水实验
| 项目 | 详情 |
|------|------|
| **常见套路** | 这个是一定要做做的，保证 24 小时以上，去楼下验收没有漏水情况，在开始进行下一步 |
| **防坑要点** | ✅ 必须 48 小时！国标最低要求<br>✅ 楼下邻居家要派人来看<br>✅ 蓄水深度≥2cm |

### 序号 30：厨房防水
| 项目 | 详情 |
|------|------|
| **常见套路** | 这个主要看厨房有没有地漏，没有地漏都不用做防水，这项就可以去掉，省一笔 |
| **防坑要点** | ✅ 有洗碗机/水槽的地方建议做<br>✅ 橱柜背后墙面也要刷防水 |

### 序号 31：水管打压
| 项目 | 详情 |
|------|------|
| **常见套路** | 在这个水路铺设完之后，一定要打压测试下，一般是半个小时 8 个压，看是否有掉压，拿纸巾检查每个接头处是否有漏水 |
| **防坑要点** | ✅ 打压值：0.8MPa 保持 30 分钟<br>✅ 压力降≤0.05MPa 为合格<br>✅ 所有接头处用纸巾擦拭检查 |

## 🪵 木工阶段（11 项）

### 序号 32：吊顶品牌
| 项目 | 详情 |
|------|------|
| **常见套路** | 问清楚吊顶所用品牌是什么，是否包括辅料费用，单层和双层分别什么价位 |
| **防坑要点** | ✅ 龙牌/可耐福/泰山石膏板<br>✅ 轻钢龙骨壁厚≥0.6mm |

### 序号 33：龙骨材料
| 项目 | 详情 |
|------|------|
| **常见套路** | 确认木龙骨还是轻钢龙骨，建议不做复杂造型的情况都用轻钢龙骨，防潮性也更好 |
| **防坑要点** | ✅ 厨卫必须用轻钢龙骨或铝合金龙骨<br>❌ 木龙骨易受潮变形 |

### 序号 34：吊顶工艺
| 项目 | 详情 |
|------|------|
| **常见套路** | 吊顶转角采用 T 字拼，拼接处挂网贴牛皮纸，摸防锈钉，细节一定要做好，不然后期很容易开裂 |
| **工艺标准** | ✅ 转角处用 L 形整板切割，避免 V 形接缝 |

### 序号 35：窗帘盒
| 项目 | 详情 |
|------|------|
| **常见套路** | 装修公司不包含窗帘盒，属于个性化项目，一般 300-500 元每个或者按照米数来收费 |
| **防坑要点** | ✅ 提前确认是否需要窗帘盒<br>✅ 含轨道还是不含要写明 |

### 序号 36：石膏线
| 项目 | 详情 |
|------|------|
| **常见套路** | 不做吊顶，可贴石膏线，主要是美观，顶面走线不直，隐藏房顶的电线，也可贴石膏线装饰，石膏线按米来收费 |
| **单价参考** | 
- PPS 石膏线：15-30 元/米
- 雕花石膏线：50-150 元/米 |

### 序号 37：木工打柜子
| 项目 | 详情 |
|------|------|
| **常见套路** | 提前和师傅确认好板材的品牌，做弧形怎么收费，胶是用的是什么品牌，让木工算算怎么切割板子更省材料，一般良心的木工师傅都会给你算的 |
| **防坑要点** | 
✅ 生态板（免漆板）: 180-280 元/㎡投影面积
❌ 现场制作不如定制柜环保和美观 |

### 序号 38：木地板铺贴
| 项目 | 详情 |
|------|------|
| **常见套路** | 木地板的收费标准是工字铺，如果要做鱼骨铺、人字铺等，又是另一笔费用 |
| **防坑要点** | ✅ 工字铺免费，异形铺法加收 30-50 元/㎡ |

### 序号 39：踢脚线
| 项目 | 详情 |
|------|------|
| **常见套路** | 这一项比较容易漏，按米收费，前期要是忘了，又是一笔不菲的人工费 |
| **单价参考** | ✅ 实木踢脚线：40-80 元/米<br>✅ MDF 踢脚线：25-50 元/米<br>✅ 不锈钢踢脚线：60-120 元/米 |

### 序号 40：门窗套
| 项目 | 详情 |
|------|------|
| **常见套路** | 按米收费，全屋门窗不少，米数也不少，提前确认要不要贴门套床套，贴什么材质的，并且确认好人工费 |
| **防坑要点** | ✅ 卧室门建议做门套（防潮）<br>❌ 卫生间门可以不做得门套 |

### 序号 41：电视背景墙
| 项目 | 详情 |
|------|------|
| **常见套路** | 可用岩板大理石或者石膏板，人工费不低，提前确认 |
| **单价参考** | ✅ 石膏板造型：150-300 元/㎡<br>✅ 岩板上墙：400-800 元/㎡ |

### 序号 42：地面保护膜
| 项目 | 详情 |
|------|------|
| **常见套路** | 主要是为了避免后期施工破坏地面瓷砖或木地板，要铺上层保护膜，一般是 10-15 元/米 |
| **防坑要点** | ✅ 必须做！保护成品<br>✅ 确认是否含在文明施工费中 |

## 🧱 瓦工阶段（16 项）

### 序号 43：地面找平
| 项目 | 详情 |
|------|------|
| **常见套路** | 铺木地板，需要提前找平地；铺瓷砖，师傅直接用水泥砂浆找平，不需要单独找平 |
| **防坑要点** | ✅ 铺地板必须找平 → 自流平 30-50 元/㎡<br>✅ 铺瓷砖由瓦工同步找平 → 确认是否收费 |

### 序号 44：美缝
| 项目 | 详情 |
|------|------|
| **常见套路** | 美缝有炫彩美缝、聚脲、环氧、瓷缝，勾缝问清楚人工费和材料费用各多少钱；用的什么品牌的美缝 |
| **单价对比** | 
- 普通美缝剂：25-45 元/米
- 真瓷胶：35-60 元/米
- 聚脲美缝：60-120 元/米 |

### 序号 45：包下水管
| 项目 | 详情 |
|------|------|
| **常见套路** | 卫生间、阳台都会有下水管，要把他们包起来才能隔音，提前确认家里有多少根管子，是单管、双管、还是三管，问清具体的收费，用红砖包还是石膏板包，要求加上隔音棉和阻尼 |
| **单价参考** | ✅ 红砖包 + 隔音棉：180-280 元/根<br>✅ 轻钢龙骨+石膏板：220-350 元/根 |

### 序号 46：外窗台坡度
| 项目 | 详情 |
|------|------|
| **常见套路** | 为了避免外窗台积水，和瓦工师傅提前沟通好，外窗台都要做好坡度，内高外高地 |
| **工艺标准** | ✅ 坡度≥5%，内高外低 1-2cm |

### 序号 47：瓷砖海棠角
| 项目 | 详情 |
|------|------|
| **常见套路** | 墙砖收口一般是贴阳角条，如果想要更美观，可做 45 度碰角，也就是海棠角，也需提前沟通是否单独加钱 |
| **单价参考** | ✅ 阳角条：15-30 元/米（便宜但有痕迹）<br>✅ 海棠角：40-80 元/米（美观但贵） |

### 序号 48：瓷砖尺寸
| 项目 | 详情 |
|------|------|
| **常见套路** | 如果选用大砖，比如 600x1200 或者 750x1500 的瓷砖，以及如果想铺贴多变型的瓷砖，是否需要增加人工费用，这个也是需提前沟通好费用的 |
| **大砖加价** | ✅ 800x800 以内：正常单价<br>✅ 750x1500 及以上：加价 30-50 元/㎡<br>✅ 岩板上墙：加价 80-150 元/㎡ |

### 序号 49：地砖上墙
| 项目 | 详情 |
|------|------|
| **常见套路** | 用地砖贴墙需要特殊工艺和粘合剂，费用会增加 |
| **单价参考** | ✅ 瓷砖胶贴墙：+20-40 元/㎡ |

### 序号 50：瓷砖通铺
| 项目 | 详情 |
|------|------|
| **常见套路** | 签合同前，需要确认瓷砖全屋通铺是否需要增加费用，通铺会比较麻烦 |
| **防坑要点** | ✅ 通铺更美观但损耗大（约 10-15%）<br>❌ 不过门石要提前确认 |

### 序号 51：过门石/窗台石
| 项目 | 详情 |
|------|------|
| **常见套路** | 这两样的铺贴费用装修公司一般不包，要提前谈好相应的费用 |
| **单价参考** | ✅ 人造石过门石：80-150 元/条<br>✅ 天然大理窗台石：300-600 元/㎡ |

### 序号 52：卫生间找坡度
| 项目 | 详情 |
|------|------|
| **常见套路** | 卫生间贴砖时一定要找坡度，做回字形地漏，确保排水顺畅 |
| **工艺标准** | ✅ 坡度 2-3%, 最低点距地漏 3-5cm |

### 序号 53：确认工艺
| 项目 | 详情 |
|------|------|
| **常见套路** | 墙压地还是地压墙，墙压地会更美观，没有朝天缝 |
| **推荐做法** | ✅ 墙压地 → 防水更好、更美观<br>❌ 地压墙 → 施工简单但不防水 |

### 序号 54：墙面拉毛
| 项目 | 详情 |

| **防坑要点** | ✅ 光滑墙面必须拉毛 → 否则会空鼓脱落<br>✅ 拉毛后用指甲刮不掉为宜 |

### 序号 55：单包套
| 项目 | 详情 |
|------|------|
| **常见套路** | 和师傅确认好门是单包套还是双包套，单包套的话需要定制门安装好以后再补贴门洞瓷砖 |
| **区别说明** | 
- 单包套：只包室内一侧（常见于有门套线的情况）
- 双包套：内外两侧都包 |

### 序号 56：瓷砖表面及缝隙

| **防坑要点** | ✅ 当天清理当天做美缝<br>❌ 水泥固化后极难清理 |

### 序号 57：壁龛
| 项目 | 详情 |

| **防坑要点** | ✅ 壁龛底部要做排水坡<br>✅ 尺寸提前规划好洗发水摆放 |

### 序号 58：贴好水电标识
| 项目 | 详情 |
|------|------|
| **常见套路** | 贴完砖以后，做好水电标识，不然后期安装五金件时一不小心把管线、水管打爆，再返工就是大工程了 |
| **防坑要点** | ✅ 要求施工方贴彩色警示贴纸<br>✅ 自己也可以用手机拍下管线走向 |

## 🎨 油漆阶段（7 项）

### 序号 59：刮腻子
| 项目 | 详情 |
|------|------|
| **常见套路** | 新房建议刮 2 次腻子，老房刮 3 次腻子，刮好之后，要打磨，注意报价单是否写清楚了数量，一定要注意，每遍干了之后在进行下一遍 |
| **工艺标准** | ✅ 第一遍：打底腻子<br>✅ 第二遍：找平腻子<br>✅ 第三遍（老房）：精细找平 |

### 序号 60：腻子品牌
| 项目 | 详情 |
|------|------|
| **常见套路** | 很多老师傅都喜欢腻子粉加胶水，这已经淘汰了，跟师傅确认好用成品腻子，直接加水就可以，南方潮湿地区用耐水腻子，强度高，防潮性更好 |
| **推荐品牌** | 
- 成品耐水腻子：美巢 800JJ、立邦腻子
- ❌ 禁止使用胶水拌腻子（甲醛超标） |

### 序号 61：墙面基层处理
| 项目 | 详情 |
|------|------|
| **常见套路** | 在刮腻子前，要对墙面进行处理，如嵌缝石膏修补线槽、墙角补阴阳角条等，提前确认是否包含了所有墙面基层处理工作，并确认具体费用；粗找平用石膏，精找平用腻子，腻子不易批刮太厚 |
| **防坑要点** | ✅ 深槽（>1cm）先用石膏填补<br>✅ 浅槽用腻子修补即可 |

### 序号 62：墙面挂网
| 项目 | 详情 |
|------|------|
| **常见套路** | 如果新房，墙面质量还可以就不需要全屋挂网，局部地方，如水电开槽，烟道地方挂网就可以了 |
| **防坑要点** | ✅ 新旧墙交界处必须挂网<br>❌ 全屋挂网多为增项套路 |

### 序号 63:乳胶漆工艺
| 项目 | 详情 |
|------|------|
| **常见套路** | 乳胶漆遵循"一底两面"的原则，一次底漆，两次面漆，底漆不能省 |
| **工艺标准** | 
✅ 一遍底漆 → 抗碱封闭
✅ 两遍面漆 → 颜色饱满均匀
❌ 只刷面漆不刷底漆 → 容易返碱发花 |

### 序号 64:乳胶漆颜色
| 项目 | 详情 |
|------|------|
| **常见套路** | 装修公司通常包 3 种浅色系，如果超过 3 种，会额外计费 |
| **单价参考** | ✅ 调色服务费：20-50 元/色 |

### 序号 65:深色乳胶漆
| 项目 | 详情 |
|------|------|
| **常见套路** | 如果涂刷深色乳胶漆，工人需要更多时间来调色和涂刷，人工费也会高 |

## 📄 报价单/合同部分（5 项）

### 序号 74:报价拆分很细
| 项目 | 详情 |
|------|------|
| **常见套路** | 本来应该是一个完整的项目，但却被拆分成好几个项目进行报价，看起来单价很低，实际却是贵的离谱;比如瓷砖的水泥砂浆费、瓷砖切割、瓷砖铺贴费本来是一个完整的项目，但它分开计费，这价格就不一样了 |
| **防坑要点** | ✅ 要求提供综合单价（含所有辅材人工）<br>✅ 对比市场总价而非单项低价 |

### 序号 75:数量乱表
| 项目 | 详情 |
|------|------|
| **常见套路** | 数量乱写是常见操作之一，目的就是为了让客户感觉到实惠，尽快签合同；比如拆墙，实际拆除是 6 平方，但报价单中写的只有 3 平方，等做完之后再给你算最终价格 |
| **防坑要点** | ✅ 自己或用激光测距仪复核尺寸<br>✅ 约定按实际发生面积结算 |

### 序号 76:材料不写品牌规格
| 项目 | 详情 |
|------|------|
| **常见套路** | 主材和辅材要提前问清品牌型号，并且签在报价单或合同中，方便材料进场验收，也避免后期发现是不理想的材料，然后再被迫升级材料 |
| **防坑要点** | ✅ 所有材料必须写明品牌 + 系列 + 型号<br>❌ "某某品牌 unspecified" = 随时可能被替换 |

### 序号 77:工艺不清楚
| 项目 | 详情 |
|------|------|
| **常见套路** | 施工工艺在报价单中要进行详细注明，明确施工工具所包含的内容，避免后期隐开增项费用，工艺也决定了工程质量 |
| **防坑要点** | ✅ 工艺说明越详细越好<br>✅ 关键工艺节点拍照留底 |

### 序号 78:质保
| 项目 | 详情 |
|------|------|
| **常见套路** | 国家强制规定水电质保 5 年，表面工程质保 2 年，注意合同上是否写清 |

✅ 基础工程：2 年
✅ 防水工程：5 年

## 📝 使用指南

### 报价审核流程：
1. **拿到报价单后，逐条对照此清单**  
   - 每一行都有对应的编号和问题点
   - 检查报价单中是否涵盖所有 80 个项目

2. **重点关注标红的"常见套路"**  
   - 这些是装修公司最容易玩的花招
   - 每一项都要问清楚并记录在案

3. **逐项确认收费标准**  
   - 哪些项目已包含？
   - 哪些需要另外收费？
   - 单价是否合理？

4. **将模糊项写入补充条款**  
   ```markdown
   【补充条款示例】
   本项目报价已包含以下 80 个常见项目（详见附件清单），
   除甲方主动提出的设计变更外，乙方不得以任何理由追加费用。
   如发现有漏项或新增项目，须经甲方书面确认后实施。
   ```

### 增项风险预警：

| 风险等级 | 项目类型 | 典型增项金额 |
|----------|----------|--------------|
| 🔴 高风险 | 水电工程 | 3,000-15,000 元 |
| 🟠 中风险 | 泥瓦工程 | 2,000-8,000 元 |
| 🟡 低风险 | 油漆工程 | 500-2,000 元 |
