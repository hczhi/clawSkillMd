### ✅ **装修公司选择 50 问清单** 
**位置**: `/home/node/clawd/skills/decoration-bidding/references/company-selection-checklist.md`

签约前必须向装修公司逐条确认的 50 个问题，涵盖六大维度：

| 类别 | 问题数 | 重点内容 |
|------|--------|----------|
| 一、公司基本情况 | 6 问 | 资质等级、自有工人 vs 外包、在建工地查看 |
| 二、设计师情况 | 4 问 | 收费模式、修改次数、不满意退款政策 |
| 三、装修费用情况 | 10 问 | 付款比例、延期赔偿、增项上限、质保承诺 |
| 四、材料情况 | 9 问 | 全包范围、环保等级、品牌型号、假货赔偿 |
| 五、施工工艺 | 18 问 | 材料品牌、验收标准、监理制度、第三方监理 |
| 六、售后情况 | 7 问 | 维修响应时间、质保期限、重大质量问题处理 |

**使用方式**: 
```
1. 打印此清单，面谈时逐项记录答案
2. 对模糊回答标记为⚠️，要求书面确认
3. 对比 3 家以上公司的答案差异
4. 将关键承诺写进合同附件
```

## 💬 使用示例

### 基础查询
```
> "卫生间防水做多高？"
→ 淋浴区墙面刷到 1.8 米，干区 30cm，门口做止水墩

> "瓷砖怎么分辨好坏？"
→ 看产地（广东砖优先）、吸水率（<0.5%）、耐磨度（莫氏 5 级以上）

> "实木地板和复合地板有什么区别？"
→ 实木：天然纹理/脚感好/娇贵怕水/价格高
   复合：耐磨耐用/便宜好打理/胶水多环保差

> "厨房台面高度应该多高？"
→ 身高÷2 + 5cm，一般 80-85cm，建议高低台设计

> "承重墙能敲吗？"
→ 绝对禁止！砸承重墙违法且危险，可能导致整栋楼成危房
```

### 方案咨询
```
> "我家 100 平，预算 15 万，能装成什么样？"
→ 我会根据面积 + 预算给出配置建议和优先级排序

> "现在是第几周了，下一步该做什么？"
→ 查看当前进度 → 对应阶段的验收清单 + 注意事项
```

### 报价分析
```
> [上传报价单图片] "帮我看看这个有没有坑"
→ OCR 识别 → 对比市场价 → 输出风险评分 + 避坑建议
```

### **新增：装修公司考察**
```
> "我要面试装修公司，有什么要问的？"
→ 提供装修公司选择 50 问清单，按 6 大类逐项询问

> "这家公司说'按实结算'是什么意思？有坑吗？"
→ 警惕！这意味着增项无上限，必须改为≤5% 的固定比例
```

## 🚀 核心心智模型

### 1️⃣ **全局观** - 把装修当成项目管理
- 进度 → 预算 → 质量 → 体验 四象限平衡
- 每个决策都有「连锁反应」
- 优先级：**隐蔽工程 > 功能性 > 美观性 > 品牌溢价**

### 2️⃣ **信息不对称破局术**
- 不迷信专业术语，翻译成「多少钱」「怎么用」
- 用「对比清单法」识别报价陷阱
- 关键节点验收标准要记在脑子里

### 3️⃣ **风险前置思维**
- 90% 的麻烦来自前期没想清楚的设计改动
- 所有口头承诺必须写进合同附件
- 预留 15-20% 的应急资金（必然会被用到😅）

### 4️⃣ **安全第一原则**
- 承重墙绝对不能动（违法 + 危险）
- 燃气/电路按规范施工
- 环保指标达标才能入住

## ⚠️ 重要提醒速查表

| 场景 | 经验法则 |
|------|---------|
| **选装修公司** | 看工地 > 看样板间 > 谈价格 |
| **装修公司考察** | 必须当面问完 50 问清单，对比 3 家以上 |
| **审核报价单** | 漏项=亏损陷阱，重点查拆旧/垃圾清运/找平 |
| **防增项** | 对照 80 个增项清单逐项核对，要求零增项承诺 |
| **防水验收** | 闭水试验至少 48 小时，楼下要来看 |
| **水电验收** | 拍照留底管线走向，水管打压 0.8MPa 保持 30 分钟 |
| **定制家具** | 先定家电尺寸再画图纸，避免后期凸出来 |
| **瓷砖选购** | 广东佛山砖优先，吸水率<0.5% |
| **窗户选择** | 临街噪音大选双层中空玻璃+Low-E 镀膜 |
| **地板保养** | 实木怕水，强化耐磨但脚感硬 |
| **插座配置** | 够用别嫌多，宁多勿少 |
| **承重墙改造** | 绝对禁止碰！违法 + 安全隐患极大 |
| **燃气管道** | 只让燃气公司专业人员操作 |
| **甲醛控制** | ENF 级板材优先，CMA 检测达标再入住 |
