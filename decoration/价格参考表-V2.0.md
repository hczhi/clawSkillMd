## 🎯 第三部分：装修公司常见套路手册

### 套路一：**低价切入，增项加价**
**表现**: 
- 报价明显低于市场价 15-30%
- 初期承诺"零增项"
- 后期不断以"必须做""行业标准"为由增加项目

**防范**:
- 对比 3 家以上正规公司报价
- 合同注明"增项不超过总价 5%"
- 所有项目逐项核对，拒绝"按实结算"

### 套路二：**材料偷梁换柱**
**表现**:
- 报价单只写"某某品牌乳胶漆"，不写具体系列型号
- 施工中声称该型号缺货，推荐更贵产品
- 使用同品牌低端系列

**防范**:
- 必须写明品牌 + 系列 + 型号 + 规格 + 颜色
- 进场时当面验收材料，拍照留存
- 约定假冒双倍赔偿

### 套路三：**水电绕线猫腻**
**表现**:
- 不按最短路径布线
- 一根管内穿过多电线（散热不良安全隐患）
- 计量时用卷尺拉直线而非沿管道

**防范**:
- 要求走顶不走地（便于检修）
- 合同约定"实测实量，误差≤10%"
- 隐蔽工程完工后请第三方监理验收并拍管线图

### 套路四：**面积虚报**
**表现**:
- 墙面面积按 2 倍建筑面积计算（实际约 2.5-3 倍）
- 地面面积忽略公摊但计入阳台等半赠送面积
- 重复计算（如吊顶边线再单独算线条）

**防范**:
- 自己或用激光测距仪复核尺寸
- 约定按实际发生面积结算

### 套路五：**拆分报价**
**表现**:
- 把一个大项拆成多个小项分别报价
- 看似单价不高，累加远超正常
- 示例：衣柜分开算板材费 + 人工费 + 五金费 + 设计费

**防范**:
- 要求提供"综合单价"（含所有辅材人工）
- 对比市场价总造价而非单项

### 套路六：**工期拖延**
**表现**:
- 合同工期写得宽松但不写延期处罚
- 项目经理同时接 3-4 个工地
- 工人随时停工待料

**防范**:
- 明确各阶段时间节点
- 约定延期每日罚款（0.1%-0.2% 工程款）
- 分期付款，尾款不少于 10%

### 套路七：**付款比例失衡**
**表现**:
- 首付 60-70%（失去制约能力）
- 中期再付 30%
- 尾款仅 5-10%

**防范**:
- 理想比例：30-40% 开工 → 30-40% 中期 → 25-30% 竣工
- 每个节点验收合格后再付款

### 套路八：**环保偷工减料**
**表现**:
- 用 E1 级板材冒充 E0 级/ENF 级
- 胶水、腻子等非板材用劣质产品
- 不清理现场灰尘直接刷漆

**防范**:
- 环保等级写进合同（E0 或 ENF）
- 完工后 CMA 检测，不合格全额退款

### 套路九：**特殊收费陷阱**
**表现**:
- 楼层增高费：无电梯楼层需增加此项费用
- 搬运中转费：1.5 吨装卸车无法直达电梯口，距离超过 20 米以上收取
- 特殊规格瓷砖加价未提前说明
- 找平厚度超标加价未明确基准

**防范**:
- 签约前确认楼层情况
- 测量搬运距离，超过 20 米需在合同中明确收费标准
- 特殊规格瓷砖加价写入合同附件
