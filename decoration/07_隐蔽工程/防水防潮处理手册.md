## 🎯 防水的核心原则

```markdown
1. **重点区域重点对待**：卫生间 > 厨房 > 阳台 > 其他区域
2. **高度要够**：淋浴区 1.8m 以上，干区 30cm
3. **细节决定成败**：管根、阴阳角、门槛石必须加强
4. **闭水试验不能省**：至少 48 小时，楼下必须确认无渗漏
```

| **阳台（无排水）** | 可不做 | - | 仅做防潮处理 |
| **地下室/半地下** | 全屋墙面 1.5m+ 地面 | 满刷 | 考虑外防水 + 内防水双重保护 |

### 选购要点

```markdown
✓ 看认证：要有中国环境标志产品认证（十环认证）、ISO9001 质量认证
✓ 看检测报告：拉伸强度≥1.2MPa，断裂伸长率≥200%（柔性防水）
✓ 看环保性：VOC 含量越低越好，优先选水性涂料
✓ 看适用性：厨卫用柔性防水，地下室用刚性防水
✗ 别贪便宜：低于 100 元的"防水涂料"大概率是假货或劣质品
```

## 🛠️ 防水施工标准流程

### 第一步：基层处理

```markdown
✅ 清理：清除表面灰尘、油污、松动的颗粒
✅ 找平：凹凸不平处用水泥砂浆找平（误差≤5mm）
✅ 湿润：施工前基面湿润但不能有明水
✅ 管根处理：下水管周边做圆弧角（半径 50mm）
✅ 阴角处理：墙角处抹成 R 角（圆弧形过渡）
```

### 第二步：细部节点加强

```markdown
📍 关键部位必须做附加层：

├─ 管根（上下水管与墙体交接处）
├─ 阴阳角（墙地交接、墙面转角）
├─ 门槛石位置（门口处）
└─ 地漏周边

做法：先刷一层涂料，然后贴无纺布或玻璃纤维网格布，再刷一层覆盖
宽度：每个方向延伸出原部位≥200mm
```

### 第三步：大面积涂刷

```markdown
刷涂要求：
├─ 涂刷方式：十字交叉法（先竖后横）
├─ 遍数：至少 2 遍，每遍垂直涂刷
├─ 间隔时间：上道实干后再刷下道（通常 4-6 小时）
├─ 厚度：单遍厚度 0.5-0.7mm，总厚度≥1.5mm
└─ 颜色判断：很多品牌防水有颜色标记，颜色均匀说明刷够厚度
```

### 第四步：养护保护

```markdown
□ 防水层固化期间禁止踩踏
□ 下一工序前做好成品保护
□ 瓷砖铺贴前检查是否有破损
```

## 💧 闭水试验详细步骤

### ⏱️ 试验时间安排

```
最佳时机：防水层完全固化后（通常施工完成后 48-72 小时）
试验时长：≥48 小时（最好 72 小时）
最佳天气：连续晴天（避免雨天影响观察）
```

### 📋 闭水试验操作步骤

```bash
1. 封堵所有出水口
   ├─ 地漏用专用堵头或塑料袋 + 沙袋封住
   ├─ 马桶下水口暂时封堵（或用临时盖子）
   └─ 浴缸/淋浴房入口用挡水条

2. 蓄水深度
   ├─ 地面蓄水深度 3-5cm
   ├─ 使用水平仪确保水面平整
   └─ 标记水位线便于观察是否渗漏

3. 第一次检查（24 小时后）
   ├─ 检查楼上自家墙面有无渗水痕迹
   ├─ 检查门口周围地面是否潮湿
   └─ 记录水位下降情况（蒸发会有少量下降）

4. 第二次检查（48-72 小时后）
   ├─ **必须去楼下邻居家检查天花板**
   ├─ 重点查看：管道周边、墙角、梁柱位置
   └─ 拍照留存证据，邻居签字确认

5. 合格标准
   ✓ 自家：无明显渗水痕迹
   ✓ 楼下：天花板干燥无水渍
   ✓ 水位：下降不超过 3mm（蒸发量）
```

### ❌ 闭水试验失败怎么办？

```markdown
问题诊断：
├─ 轻微渗水 → 可能是防水层有微小裂缝
│   └─ 处理：清理干净后重新涂刷一道，再次闭水试验
│
├─ 明显渗漏 → 可能有孔洞或未涂刷到位
│   └─ 处理：找出漏点，局部加强后整面重做
│
└─ 持续渗漏 → 防水层失效严重
    └─ 处理：铲除重做，注意基层处理和节点加强
```

## ⚠️ 防水工程常见错误案例

### ❌ 错误 1：淋浴区防水高度不够
**问题描述**：淋浴区只刷到 1.2m，洗澡时水会顺着墙体流到隔壁卧室  
**正确做法**：淋浴区墙面防水至少 1.8m，有条件建议刷到顶

### ❌ 错误 2：门没做止水坎
**问题描述**：卫生间地面比客厅低，但门口没有水泥台，水会漫出去泡坏地板  
**正确做法**：门口做 5cm 高的止水墩，瓷砖铺在上面

### ❌ 错误 3：管根没做加强
**问题描述**：下水管周边防水不严密，从根部开始渗漏  
**正确做法**：管根做圆弧角 + 附加层，宽度≥200mm

### ❌ 错误 4：闭水试验时间不足
**问题描述**：只做了 24 小时就贴砖，第 3 天才发现漏水  
**正确做法**：闭水试验至少 48 小时，有条件做 72 小时

### ❌ 错误 5：没通知楼下业主
**问题描述**：自己觉得没问题就贴砖了，入住后被投诉漏水才发现问题  
**正确做法**：闭水试验必须叫楼下业主上门查看并签字确认

### ❌ 错误 6：防水层被破坏
**问题描述**：闭水合格后贴砖时把防水层戳破了  
**正确做法**：贴砖时穿软底鞋，轻拿轻放，必要时二次补刷

### ❌ 错误 7：用错防水材料
**问题描述**：地下室用了柔性防水，时间久了开裂渗漏  
**正确做法**：地下室、阳台等无变形风险区域用刚性防水，厨卫用柔性防水

## 🏗️ 特殊情况的防水处理

### 1. 老房翻新防水

```markdown
挑战：原有防水层可能老化失效
解决思路：
├─ 铲掉原有面层到基层
├─ 检查基层是否有裂缝，如有需修补
├─ 重新做全套防水流程
└─ 建议整体重做而非局部修补
```

### 2. 卫生间改干湿分离

```markdown
关键点：
├─ 湿区地面要找坡，坡度 1%-2% 朝向地漏
├─ 隔断下方要做止水坎
├─ 玻璃胶要选防霉型
└─ 地漏要选深水封式防臭
```

### 3. 阳台改造为洗衣区

```markdown
防水重点：
├─ 洗衣机位置地面全刷防水
├─ 墙面刷到 30cm 高
├─ 排水管要做存水弯
└─ 预留溢水口防止倒灌
```

## 🛡️ 防潮处理补充方案

### 南方地区特别注意事项

```markdown
【梅雨季防潮】
✓ 地面瓷砖缝隙做美缝（比普通勾缝更防潮）
✓ 衣柜背板留空隙或使用防潮板材
✓ 墙面腻子选用耐水腻子
✓ 安装除湿机或新风系统

【 basement 防潮】
✓ 外墙做外防水（最有效）
✓ 室内再做内防水作为双保险
✓ 地面做架空层或加隔汽膜
✓ 保持通风，安装排风扇
```
