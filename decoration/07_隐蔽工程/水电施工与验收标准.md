### 阶段二：开槽布管施工标准

#### 🔧 材料规格要求

**电线规格对照表**
| 使用场景 | 电线截面 | 颜色规范 |
|---------|---------|---------|
| 照明线路 | 1.5㎡ | 红色/绿色/蓝色 |
| 普通插座 | 2.5㎡ | 红色/黄色/蓝色 |
| 空调/厨房插座 | 4㎡ | 红色/黄色/蓝色 |
| 大功率电器（电地暖/中央空调） | 6㎡ | 红色 |
| 进户主线 | 10-16㎡ | 根据实际负荷确定 |

**水管规格要求**
| 使用场景 | 管径 | 壁厚 | 压力等级 |
|---------|------|------|---------|
| 冷热水管 | DN20（6 分） | ≥2.8mm | PN25 以上 |
| 支管（水龙头） | DN15（4 分） | ≥2.3mm | PN25 以上 |
| 排水主管 | DN110 | 符合国标 | - |
| 排水支管 | DN50 | 符合国标 | - |

#### 📐 施工工艺标准

**强电布线规范**
```markdown
✅ 开槽深度：管径 + 10-15mm 保护层
✅ 开槽宽度：两管并排 = 管径之和 + 20mm
✅ 横平竖直：地面走直线，墙面横平竖直（便于后期打钉）
✅ 强弱电分离：间距≥30cm，交叉处包锡箔纸防干扰
✅ 转弯半径：弯角处要用弹簧弯管，弧度平缓（方便换线）
✅ 管内穿线：同一根管不超过 3 根线，截面积≤40%
✅ 接线方式：所有接头必须在接线盒内，禁止直埋
✅ 固定间距：PVC 管卡固定，间距≤80cm
```

**弱电布线规范**
```markdown
✅ 网线类型：六类线（Cat6）或以上，支持千兆网络
✅ 每房间至少 1 个网口，优先全屋千兆
✅ 弱电箱尺寸要够大（至少 60×40×20cm），放路由器和光猫
✅ 弱电和强电平行间距≥30cm，交叉处必须包锡箔纸
✅ 网线两端贴标签（标明房间用途，如"主卧网"）
```

**水路布设规范**
```markdown
✅ 走顶不走地：漏水易发现且维修不破坏地板
✅ 左热右冷：两个水管间距 15cm，中心距一致
✅ 上热下冷：热水管在上方，防止热量向下传导损失
✅ 卡扣固定：管卡间距≤60cm，转角处必装
✅ 打压测试：安装完成后必须进行（详见验收部分）
✅ 保温处理：热水管包保温棉，减少热量损失
✅ 存水弯：排水管必须做存水弯防臭
```

### 阶段三：电路验收标准

#### 🔌 目测检查清单

```markdown
□ 所有线管固定牢固，无松动脱落
□ 管内穿线顺畅，抽拉活动自如
□ 接线盒位置合理，盖板安装后不遮挡开关面板
□ 强弱电交叉处已包锡箔纸
□ 电线颜色规范统一（火线红、零线蓝、地线黄绿双色）
□ 所有接头在接线盒内，无裸露导线
□ 墙面开槽深度足够，抹灰后不会开裂
```

#### 📊 电气测试项目

**1. 通断测试**
```bash
工具：万用表
方法：
1. 关闭总闸，拆下待测回路的零线和火线
2. 用万用表电阻档测量回路两端
3. 正常情况：电阻接近 0Ω（通路）
异常：电阻无穷大（断路）
```

**2. 绝缘电阻测试**
```bash
工具：兆欧表（摇表）
标准：相线对地绝缘电阻≥0.5MΩ
步骤：
1. 断开所有电器设备
2. 分别测试火线/零线对地绝缘
3. 记录数值，低于标准需排查原因
```

**3. 相位检测**
```bash
工具：相位检测仪
标准：左零右火上接地
检测项：
✓ L（火线）接正确位置
✓ N（零线）接正确位置
✓ PE（地线）连接可靠
```

**4. 漏电保护测试**
```bash
工具：漏电保护测试仪
步骤：
1. 按下漏电保护器的"T"测试按钮
2. 应立即跳闸断开电源
3. 复位后重新合闸
注意：每月测试一次确保有效
```

### 阶段四：水路验收标准

#### 💧 目测检查清单

```markdown
□ 水管布局整齐，横平竖直
□ 管卡固定牢固，无松动
□ 热水管包裹保温棉
□ 阀门位置便于操作
□ 出水口高度符合设计规范
□ 无渗漏水痕迹
```

#### 📊 水压测试流程（关键！）

```bash
测试工具：手动打压泵（压力表量程 0-1.6MPa）

测试标准：
├─ 试验压力 = 工作压力的 1.5 倍（通常 0.8-1.0MPa）
├─ 稳压时间：≥30 分钟
└─ 压降标准：≤0.05MPa 为合格

操作步骤：
1. 封闭所有出水口
2. 缓慢加压至试验压力
3. 观察压力表 30 分钟
4. 压降≤0.05MPa 且无渗漏为合格
5. 不合格立即查找漏点修复后重测

注意事项：
✗ 冬季施工要在 5℃以上进行，防止冻裂
✗ 试压期间保持压力稳定，不要频繁开关阀门
✗ 拍照记录试压初始压力值作为证据
```

#### 🎯 排水系统验收

```markdown
□ 排水管坡度符合要求（DN50 管坡度 2%，DN110 管坡度 1%）
□ 所有排水口畅通无阻
□ 存水弯密封良好，无返味
□ 地漏安装平整，坡度朝向地漏
□ 闭水试验合格（详见防水章节）
```

## ⚠️ 水电改造常见错误案例

### ❌ 错误 1：插座位置被家具挡住
**问题描述**：床头插座距离地面只有 30cm，床头柜挡住了无法充电  
**正确做法**：床头插座应高于床头柜 15-20cm，或选用带 USB 的多功能插座

### ❌ 错误 2：厨房插座不够用
**问题描述**：台面上只有 2 个插座，买了电饭煲就没位置插榨汁机了  
**正确做法**：厨房台面预留 4-5 个带开关的五孔插座，避免拔插插头

### ❌ 错误 3：空调插座高度不对
**问题描述**：空调插座装在柜子后面，外机线从柜子里穿过太丑  
**正确做法**：挂机空调插座高度 2.2-2.4m，立柜式空调插座高度 30-40cm

### ❌ 错误 4：网线选得太差
**问题描述**：用的 Cat5 网线只支持百兆，浪费宽带提速  
**正确做法**：全屋用 Cat6 六类线，未来升级到千兆网络没问题

### ❌ 错误 5：水管走地不保温
**问题描述**：冬天热水管热量散失快，而且漏水发现不了  
**正确做法**：水管走顶，热水管包保温棉，冷水管也可包一层防潮

### ❌ 错误 6：强弱电混在一起
**问题描述**：网线旁边紧挨着电源线，网速慢、掉线频繁  
**正确做法**：强弱电间距≥30cm，交叉处用锡箔纸包裹隔离

### ❌ 错误 7：没留检修口
**问题描述**：暗装阀门坏了，砸墙才能修  
**正确做法**：中央空调管道阀门口、水管阀门处设置可开启检修口

### 建议请专业师傅的情况

```markdown
✓ 从未接触过水电改造的新手
✓ 没有专业工具和仪表
✓ 时间紧张不想折腾
```

```markdown

□ 简单的故障排查（如跳闸、短路）
```
