## 🎯 吊顶的核心功能

```markdown
【主要作用】
1️⃣ 遮挡管线（空调管道、电线管）
2️⃣ 隐藏设备（中央空调、新风系统）
3️⃣ 营造层次（无主灯设计必备）
4️⃣ 隔音保温（附加功能）
5️⃣ 修饰空间（拉高层高、平衡视觉）

❌ 不要盲目跟风：层高 <2.7m 慎做复杂吊顶！
```

### 1️⃣ 石膏板吊顶详解

#### 龙骨结构

```markdown
【轻钢龙骨 vs 木龙骨】
├─ 轻钢龙骨（强烈推荐）
│   ├─ 优点：防火防潮、不变形
│   └─ 适用：所有区域
   
└─ 木龙骨（不推荐）
    ├─ 问题：易受潮变形、有防火隐患
    └─ 仅适用：局部小范围造型

【规格选择】
├─ 主龙骨：C50/C60/C70（数字代表高度）
├─ 副龙骨：T 型或 C 型
└─ 吊杆：φ8 钢筋（间距≤100cm）
```

---

#### 石膏板类型

| 类型 | 厚度 | 用途 | 备注 |
|------|------|------|------|
| **普通石膏板** | 9.5mm | 干燥区吊顶 | 卧室客厅通用 |
| **耐水石膏板** | 9.5-12mm | 靠近卫生间区域 | 白色/绿色标识 |
| **防火石膏板** | 9.5-12mm | 有防火要求区域 | 红色标识 |
| **双层石膏板** | 9.5+9.5mm | 大跨度吊顶 | 防开裂加强 |

#### 关键节点工艺

##### A. 转角 L 型整板 ✅ 必做！

```markdown
【为什么重要】
→ 两个石膏板拼接处最容易开裂

【正确做法】
├─ 转角处用整块石膏板切割成"L"型
├─ 禁止两块板直接拼接在角部
└─ 效果：大幅降低开裂概率

【示意图】
错误 ❌        正确 ✅
  ╱│            ___
 / ╲           /   \
╱___╲          │    │
              ───────
```

---

##### B. V 型缝处理

```markdown
【目的】
→ 便于腻子填补，避免裂缝外显

【操作步骤】
├─ 两块石膏板接缝处预留 5-8mm 缝隙
├─ 用电工刀切出 V 型槽
├─ 填嵌缝石膏 + 贴绷带
└─ 最后刮腻子找平
```

---

##### C. 伸缩缝设置

```markdown
【何时需要】
├─ 吊顶长度超过 12m
├─ 大面积吊顶分区处理
└─ 不同材质交接处

【做法】
└─ 预留 10-15mm 缝隙，后期用装饰条覆盖
```

#### 常见吊顶形式

##### ① 双眼皮吊顶

```markdown
【特点】
├─ 两层石膏板错层设计
├─ 简单耐看、不压层高
└─ 适合现代简约风格

【尺寸参考】
├─ 第一层下吊：10-12cm
├─ 第二层下吊：6-8cm
├─ 宽度：30-40cm
└─ 总厚度：差值 4-6cm

【优点】
✓ 比平顶省钱
✓ 比单顶有层次
✓ 藏筒灯射灯方便
```

---

##### ② 无边框平顶（无主灯）

```markdown
【特点】
├─ 完全封闭吊顶
├─ 嵌入式灯具整齐排列
└─ 视觉效果极简干净

【下吊高度】
├─ 普通：12-15cm（藏灯具即可）
├─ 含中央空调：25-35cm
└─ 含新风系统：30-40cm

【注意事项】
✗ 成本高（材料 + 人工都贵）
✗ 压层高明显
✗ 需要全屋布线预埋
```

---

##### ③ 局部吊顶

```markdown
【适用场景】
✓ 餐厅区吊顶（遮梁）
✓ 走廊吊顶（遮管线）
✓ 电视背景墙上方（藏投影幕布）

【优点】
→ 不压整体层高
→ 针对性解决问题
→ 性价比高
```

### 2️⃣ 铝扣板吊顶详解（厨卫专用）

#### 规格与厚度

```markdown
【常规规格】
├─ 300×300mm（小格，老式）
├─ 300×600mm（中等，主流）
└─ 400×400mm / 600×600mm（大板，新趋势）

【厚度选择】
├─ 国标：0.6mm（合格线）
├─ 推荐：0.7-0.8mm（家用耐用）
└─ 高端：1.0mm+（商用级别）

⚠️ 警惕："加厚"铝扣板很多是回收料，别贪便宜！
```

### 3️⃣ 微水泥吊顶（奢侈选项）

```markdown
【什么是微水泥】
→ 新型装饰材料，可上墙地上顶，实现无缝一体化

【优点】
✓ 无缝效果超高级
✓ 耐磨耐刮
✓ 防水防霉（厨卫可用）
✓ 色彩丰富可选

【缺点】
✗ 价格极高（200-500 元/㎡）
✗ 对基层要求极高（必须找平）
✗ 容易开裂（施工难度大）
✗ 维修困难（补色难匹配）

【适用人群】
→ 预算充足、追求极致设计的豪宅用户

❌ 不建议：预算有限、中小户型慎选
```

## 💡 吊顶施工流程

### 石膏板吊顶步骤

```bash
1. 弹线定位
   └─ 标记吊顶高度、灯具位置

2. 安装吊杆
   └─ 间距≤100cm，膨胀螺丝固定

3. 安装主龙骨
   └─ C 型轻钢龙骨，调平

4. 安装副龙骨
   └─ T 型或 C 型，间距 30-40cm

5. 安装石膏板
   └─ 自攻螺丝固定，钉眼点防锈漆

6. 接缝处理
   └─ V 型缝 → 嵌缝石膏 → 绷带 → 腻子

7. 打磨刮腻子
   └─ 2-3 遍，打磨平整

8. 刷底漆 + 面漆
   └─ 一底两面，阴阳角直顺
```

---

### 铝扣板吊顶步骤

```bash
1. 测量放线
   └─ 确定水平线和灯位

2. 安装边龙骨
   └─ 沿墙四周打膨胀螺丝

3. 安装吊杆和主龙骨
   └─ 间距 90-100cm

4. 安装三角龙骨
   └─ 扣板卡入龙骨

5. 安装电器模块
   └─ 灯具、浴霸、换气扇嵌入

6. 安装铝扣板
   └─ 从中间向两边扣合

7. 打胶收边
   └─ 墙角缝隙用防霉玻璃胶
```

## 📐 吊顶高度控制标准

### 客厅/卧室

```markdown
【标准下吊】
├─ 不装中央空调：10-15cm
├─ 装中央空调：28-35cm
└─ 层高 <2.7m：建议不做吊顶或用明装灯具

【层高计算】
成品高度 = 原始层高 - 下吊高度

举例：
├─ 原层高 2.8m，下吊 12cm → 成品 2.68m
├─ 原层高 2.7m，下吊 30cm → 成品 2.4m（压抑感明显）
```

---

### 厨房/卫生间

```markdown
【最低高度】
├─ 吊顶底部距地 ≥2.2m
├─ 淋浴花洒顶部距吊顶 ≥20cm
└─ 燃气热水器下方距吊顶 ≥15cm

【特殊设备预留】
├─ 浴霸：提前确认开孔尺寸（295×295mm）
├─ 凉霸：集成吊顶专用模块
└─ 新风管道：提前规划走线
```

## 🚫 吊顶常见错误与避坑

### ❌ 错误 1：层高太低强行吊顶

```markdown
【症状】
→ 做完后感觉房间像山洞

【原因】
→ 没有提前测量原始层高

【补救】
├─ 放弃复杂吊顶
├─ 改用明装射灯/轨道灯
└─ 局部吊顶代替全屋吊顶
```

---

### ❌ 错误 2：没做 L 型转角

```markdown
【症状】
→ 入住半年后墙角出现裂纹

【原因】
→ 两块石膏板直接拼在角部

【修复】
├─ 铲除裂缝处腻子
├─ 切开 V 缝重做
└─ 贴玻璃纤维网格布加强
```

---

### ❌ 错误 3：龙骨间距过大

```markdown
【症状】
→ 吊顶轻微下沉、不平

【原因】
→ 副龙骨间距超过 40cm

【预防】
└─ 严格按标准施工，验收时拿尺子量
```

---

### ❌ 错误 4：厨卫不用防霉胶

```markdown
【症状】
→ 墙角发黑发霉

【原因】
→ 用了普通玻璃胶

【正确做法】
└─ 厨卫必须用防霉玻璃胶（虽然贵但值得）
```
