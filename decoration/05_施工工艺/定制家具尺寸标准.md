## 🎯 定制家具设计的核心原则

```markdown
【基本原则】
1️⃣ 人体工学优先（好用 > 好看）
2️⃣ 功能分区合理（拿取方便）
3️⃣ 预留电器尺寸（提前规划）
4️⃣ 避免死角浪费（空间利用率）
5️⃣ 五金配置到位（寿命长、体验好）
```

## 🍳 橱柜尺寸标准

### 整体尺寸

```markdown
【地柜高度计算】
身高 ÷ 2 + 5cm = 舒适操作高度

举例：
├─ 身高 160cm → 85cm
├─ 身高 170cm → 90cm
└─ 身高 180cm → 95cm

【标准范围】
├─ 最低：80cm（小户型通用）
├─ 推荐：82-88cm（适配大多数人）
└─ 最高：90-95cm（高个子专属）

💡 建议：如果夫妻身高差大，用"高低台"设计
   └─ 水槽区比灶台区高 8-10cm（洗菜不弯腰，炒菜不架胳膊）
```

---

### 各区域详细尺寸

#### A. 地柜部分

| 项目 | 尺寸要求 | 说明 |
|------|---------|------|
| **台面深度** | 60cm | 标准深度，太深容易碰头 |
| **地柜进深** | 55-60cm | 考虑抽屉厚度 |
| **吊柜深度** | 30-35cm | 低于地柜 25-30cm，防碰头 |
| **吊柜高度** | 距离台面 60-70cm | 太高够不着，太低挡视线 |
| **吊柜离地高度** | 150-160cm | 视使用者身高调整 |

---

#### B. 功能分区尺寸

##### 水槽区

```markdown
【单槽方案】
├─ 长度：≥70cm
├─ 宽度：≥45cm
└─ 深度：≥20cm

【双槽方案】
├─ 左槽（主洗）：≥45×40cm
├─ 右槽（沥水/副洗）：≥30×35cm
└─ 总长度：≥85cm

【龙头位置】
└─ 居中或偏一侧（根据水槽设计）
```

##### 灶台区

```markdown
【燃气灶尺寸】
├─ 标准开孔：65×40cm（需确认具体型号）
├─ 左右留空：每侧≥10cm（防热变形）
└─ 后方距墙：≤5cm（贴壁稳定）

【油烟机下方】
├─ 底部离台面：65-75cm（吸烟效果好）
└─ 两侧距吊柜：≥15cm（散热空间）
```

##### 切配区（黄金操作区）

```markdown
【理想长度】
└─ ≥80cm（备菜需要足够空间）

【必备配置】
✓ 墙面挂杆（放刀具、铲勺）
✓ 台面边缘沥水条
✓ 插座（料理机、电饭煲等小家电）
```

#### C. 抽屉与拉篮

| 类型 | 宽度 | 深度 | 高度 | 用途 |
|------|------|------|------|------|
| **浅抽** | 60cm | 50cm | 12-15cm | 餐具、调料、零食 |
| **中抽** | 60cm | 50cm | 18-22cm | 碗盘、锅具 |
| **深抽** | 60cm | 50cm | 25-30cm | 囤货、重物 |
| **调味拉篮** | 20-30cm | 50cm | 50cm | 瓶瓶罐罐集中收纳 |
| **转角飞碟** | 80-90cm | 适用转角 | - | 利用死角空间 |
| **小怪物拉篮** | 60-80cm | 适用转角 | - | 大容量转角收纳 |

### ⚡ 电路规划

```markdown
【插座布置】

台面以上（带开关五孔）：
├─ 数量：4-6 个
├─ 高度：距台面 30-40cm
└─ 间距：均匀分布，避免被厨具遮挡

水槽下方（防溅盒）：
├─ 净水器插座 × 1
├─ 垃圾处理器插座 × 1
├─ 洗碗机插座 × 1
└─ 小厨宝插座（可选）× 1

冰箱专用回路：
└─ 位置：靠墙角，独立开关控制

燃气热水器：
└─ 位置：吊柜内或吊顶上方，距地面 180-200cm
```

## 🛋️ 餐边柜尺寸标准

### 整体布局

```markdown
【常见形式】
├─ 满墙柜（餐厅整面墙）
├─ 局部柜（靠墙一侧）
└─ 隔断柜（客餐厅分隔）

【标准尺寸】
├─ 总高度：220-240cm（顶天立地）
├─ 总宽度：根据实际墙面定制
├─ 深度：35-40cm（不要太深，占地）
└─ 底部悬空：35-45cm（放水壶、咖啡机）
```

---

### 功能分区

#### A. 中间留空区

```markdown
【尺寸标准】
├─ 高度：50-60cm（放下所有小家电）
├─ 深度：35cm（与柜体一致）
└─ 长度：根据需求 80-200cm

【配置建议】
✓ 轨道插座（方便插拔小家电）
✓ 感应灯带（晚上使用更方便）
✓ 背板留缝（电线走线）

【适合放置物品】
├─ 饮水机/茶吧机
├─ 咖啡机
├─ 面包机
├─ 榨汁机
└─ 消毒柜（迷你款）
```

---

#### B. 上部吊柜

```markdown
【尺寸】
├─ 每层高度：30-40cm
├─ 深度：30-35cm
└─ 门板高度：60-70cm（视觉协调）

【功能】
└─ 存放不常用物品（节日餐具、干货杂粮）
```

---

#### C. 下部地柜

```markdown
【抽屉配置】
├─ 浅抽屉 × 2-3 个（餐具、纸巾）
├─ 中抽屉 × 1-2 个（杯具、茶具）
└─ 推拉篮 × 1 个（囤货杂物）

【柜门配置】
└─ 高柜门 × 1-2 个（大型电器或囤货）
```

## 🚪 玄关鞋柜尺寸标准

### 基础尺寸

```markdown
【深度】
└─ 35cm（常规鞋刚好，加靴子需 40cm）

【高度】
├─ 通顶：220-240cm
└─ 非通顶：120-150cm（半截柜）

【宽度】
└─ 根据玄关实际尺寸定制
```

---

### 内部结构

#### A. 底部悬空区

```markdown
【高度】15-20cm（常穿鞋直接踢进去）

【配置建议】
✓ 感应灯带（晚上回家不摸黑）
✓ 透气孔（通风除臭）
✓ 换鞋凳（可选，深度 40cm，高度 45cm）
```

---

#### B. 中间留空区

```markdown
【高度】35-45cm（放钥匙、包包、快递）

【配置建议】
✓ 插座 × 1（手机充电、小夜灯）
✓ 感应灯带
✓ 洞洞板（挂帽子、围巾）
✓ 挂钩 × 3-5 个（包、伞、外套）
```

---

#### C. 鞋格设计

```markdown
【单层高度】15-18cm（常规运动鞋、皮鞋）

【分层数量】
├─ 低帮鞋位：10-12 层 → 约 40-50 双
└─ 加高靴位：2-3 层（30cm 高）→ 长靴 4-6 双

【可调层板】
└─ 建议 3-4 层做活动层板（适应不同鞋型）
```

## 💡 常见问题速查

### Q1: 橱柜高度到底怎么定？

```markdown
【单人家庭】
→ 直接用公式：身高÷2 + 5cm

【双人家庭（身高差异大）】
→ 用身高较高的一方计算，或者采用高低台设计：
   ├─ 水槽区 = 矮个子身高÷2 + 5cm
   └─ 灶台区 = 水槽区高度 - 8~10cm

【试错方法】
1. 站在橱柜前模拟动作
2. 最舒服的高度是：手臂自然下垂时手肘下方 10cm
```

### Q2: 衣柜深度不够怎么办？

```markdown
【现状】
→ 有些户型只能做到 55cm 深

【解决方案】
1. 改用侧开门衣柜（非推拉门）
2. 挂衣杆改为斜向安装（节省 5-8cm）
3. 压缩衣架厚度（选窄肩衣架）
4. 部分挂衣改为叠放为主

❗ 注意：60cm 才是标准深度，少于 55cm 真的很尴尬
```

### Q3: 鞋柜做多少层合适？

```markdown
【普通家庭】
→ 10-12 层（容纳全家日常鞋子）

【有小孩家庭】
→ 增加 2-3 层儿童鞋位

【特殊情况】
├─ 靴子多 → 专设 2-3 层高靴位（30cm 高）
├─ 运动鞋多 → 增加普通层数（15-18cm 高）
└─ 拖鞋多 → 底部留开放区专门放
```
