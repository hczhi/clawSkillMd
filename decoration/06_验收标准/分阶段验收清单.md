# ✅ 装修分阶段验收清单

> _每个节点怎么验收？照着检查不踩坑_

---

## 🎯 验收的核心原则

```markdown
1️⃣ 隐蔽工程必须到场验收（水电/防水）
2️⃣ 关键材料进场要确认
3️⃣ 拍照留底所有管线走向
4️⃣ 发现问题当场提出整改
5️⃣ 验收合格再付款
```

---

## 🔌 一、水电改造验收

### 验收时机
- 电线穿完、水管铺设完成后
- 封槽之前

---

### 强电验收清单

| 检查项 | 标准要求 | 检查方法 |
|--------|---------|---------|
| **线管固定** | 管卡间距≤80cm | 目测 + 手拉测试 |
| **强弱电分离** | 间距≥30cm | 卷尺测量 |
| **交叉处防护** | 包锡箔纸防干扰 | 目测 |
| **管内穿线** | 截面积≤40% | 手能轻松抽动 |
| **接线盒位置** | 便于后期面板安装 | 目测 |
| **电线颜色规范** | 红/黄/蓝（火线）| 目测 |
| **地线黄绿双色** | 统一标识 | 目测 |
| **接头在接线盒内** | 无裸露导线 | 目测 |

---

### 电路测试项目

```bash
【通断测试】
工具：万用表
标准：电阻接近 0Ω（通路）
异常：电阻无穷大 → 断路

【绝缘电阻测试】
工具：兆欧表
标准：相线对地绝缘电阻 ≥0.5MΩ
操作：
├─ 断开所有电器
├─ 分别测火线/零线对地绝缘
└─ 记录数值低于标准需排查原因

【相位检测】
工具：相位检测仪
标准：左零右火上接地
检查项：
✓ L（火线）接正确
✓ N（零线）接正确  
✓ PE（地线）连接可靠

【漏电保护测试】
工具：漏电保护测试仪
操作：按下"T"测试按钮 → 应立即跳闸
频率：每月测试一次
```

---

### 水路验收清单

| 检查项 | 标准要求 | 检查方法 |
|--------|---------|---------|
| **水管布局** | 横平竖直、整齐美观 | 目测 |
| **管卡固定** | 间距≤60cm | 手推拉测试 |
| **热水管保温** | 包裹保温棉 | 目测 |
| **阀门位置** | 便于操作维修 | 目测 |
| **出水口高度** | 符合设计规范 | 卷尺测量 |
| **无渗漏水痕迹** | 干燥无水渍 | 目测 |

---

### 水压测试（关键步骤）⭐

```bash
测试工具：手动打压泵（压力表量程 0-1.6MPa）

【测试标准】
试验压力：0.8-1.0MPa（工作压力的 1.5 倍）
稳压时间：≥30 分钟
压降标准：≤0.05MPa 为合格

【操作步骤】
1. 封闭所有出水口
2. 缓慢加压至试验压力
3. 观察压力表 30 分钟
4. 压降≤0.05MPa 且无渗漏 → 合格
5. 不合格 → 查找漏点修复后重测

【注意事项】
✗ 冬季施工要在 5℃以上进行（防冻裂）
✗ 试压期间保持压力稳定
✗ 拍照记录初始压力值作为证据
```

---

### 排水系统验收

```markdown
□ 排水管坡度符合要求
   ├─ DN50 管坡度 2%
   └─ DN110 管坡度 1%

□ 所有排水口畅通无阻
   └─ 倒水测试无堵塞

□ 存水弯密封良好
   └─ 无返味现象

□ 地漏安装平整
   └─ 坡度朝向地漏
```

## 💧 二、防水工程验收

### 验收时机
- 防水涂料干透后（通常施工完成 48-72 小时）
- 贴砖之前

---

### 目测检查清单

| 检查项 | 标准要求 |
|--------|---------|
| 基层已清理干净 | 无灰尘、油污 |
| 找平到位 | 凹凸不平已修补 |
| 管根做圆弧角 | 半径 50mm |
| 阴角做 R 角 | 圆弧形过渡 |
| 细部节点加强层 | 宽度≥200mm |
| 涂刷均匀无遗漏 | 颜色一致 |
| 涂层厚度达标 | ≥1.5mm |

---

### 闭水试验流程（必须执行！）⭐

```markdown
【最佳时机】
→ 防水层完全固化后（48-72 小时）

【试验时长】
→ ≥48 小时（最好 72 小时）

【操作步骤】
1. 封堵所有出水口
   ├─ 地漏用专用堵头或沙袋
   ├─ 马桶下水临时封堵
   └─ 淋浴房入口用挡水条

2. 蓄水深度
   ├─ 地面蓄水 3-5cm 深
   ├─ 使用水平仪确保水面平整
   └─ 标记水位线便于观察

3. 第一次检查（24 小时后）
   ├─ 检查自家墙面有无渗水
   ├─ 检查门口周围是否潮湿
   └─ 记录水位下降情况

4. 第二次检查（48-72 小时后）
   ├─ **必须去楼下邻居家查看天花板**
   ├─ 重点部位：管道周边、墙角、梁柱
   └─ 拍照留存证据，邻居签字确认

【合格标准】
✓ 自家：无明显渗水痕迹
✓ 楼下：天花板干燥无水渍
✓ 水位：下降不超过 3mm（蒸发量）
```

## 🧱 三、泥工铺贴验收

### 验收时机
- 瓷砖铺贴完成后 24 小时以上
- 美缝施工前

---

### 空鼓率检查

```bash
【检查工具】
空鼓锤（或小铁锤）

【检查范围】
├─ 客厅地砖：抽检 30%
├─ 厨卫墙砖：全数检查
└─ 重点区域：瓷砖四角和中心

【标准】
├─ 单块砖空鼓面积 < 15%
├─ 整体空鼓率 < 5%
└─ 中心区域严禁空鼓

【处理方式】
发现问题立即铲掉重贴
```

---

### 平整度检查

```bash
【工具】
2 米靠尺 + 塞尺

【标准】
├─ 表面平整度误差 ≤3mm
├─ 接缝高低差 ≤0.5mm
└─ 对角线长度差 ≤3mm

【检查方法】
1. 将靠尺贴在瓷砖表面
2. 看缝隙大小
3. 用塞尺测量
```

---

### 坡度检查（卫生间/阳台）

```bash
【检查项】
地漏四周是否有明显积水

【标准】
├─ 坡度 1%-2%（每米下降 1-2cm）
└─ 倒水后能迅速排走，无积水

【简单测试法】
往地面倒一盆水 → 观察流向和速度
```

## 🪵 四、木工吊顶验收

### 验收时机
- 腻子刮完、打磨完成后
- 刷漆之前

---

### 检查清单

| 检查项 | 标准要求 |
|--------|---------|
| 龙骨间距 | ≤40cm |
| 转角 L 型整板 | 必做！防止开裂 |
| V 型缝处理 | 接缝处切 V 形槽 |
| 自攻螺丝防锈 | 点涂防锈漆 |
| 石膏板接缝 | 紧密无缝隙 |
| 表面平整度 | ≤3mm/2m |
| 阴阳角顺直 | 目测无明显弯曲 |

## 🎨 五、油漆墙面验收

### 验收时机
- 面漆全部完工至少 7 天后
- 入住之前

---

### 墙面质量检查

```markdown
【目测检查】
□ 表面平整光滑，无颗粒、流坠
□ 无色差、透底现象
□ 无开裂、起皮、脱粉
□ 阴阳角垂直顺直
□ 踢脚线上沿干净无污渍

【手摸检查】
□ 不掉粉、不掉色
□ 触感细腻平整

【灯光侧照检查】
□ 无明显波浪纹、接痕
□ 无明显砂纸痕

【敲击检查】
□ 无空鼓声（区别于瓷砖）
```

---

### 涂料用量核对

```markdown
【理论用量计算】
刷漆面积 ≈ 建筑面积 × 2.5
每升乳胶漆 ≈ 可刷 12-14㎡（一遍）
实际两遍用量 ≈ 建筑面积 × 0.15L/㎡

举例：
100㎡房子 → 预计用漆 15L → 一般买 2 桶 5L + 1 桶 5L
```

## 🛋️ 八、地板安装验收

### 验收时机
- 地板铺贴完成后 24 小时内

---

### 检查清单

| 检查项 | 标准要求 |
|--------|---------|
| 表面平整 | 无起伏感 |
| 踩踏无异响 | 行走无咯吱声 |
| 缝隙均匀 | 板间≤0.5mm |
| 收边条稳固 | 无翘起松动 |
| 与墙面留缝 | 8-12mm（伸缩缝） |
| 踢脚线贴合 | 与墙面无缝隙 |
| 色差在允许范围内 | 无明显深浅差异 |
| 表面完好 | 无划痕、磕碰 |

---

### 空鼓检查（实木地板）

```bash
【检查方法】
拿小木锤轻敲地板不同位置

【标准】
├─ 无声响 → 粘贴牢固
└─ 有清脆回响 → 有空鼓需整改
```

## 🔦 九、灯具开关验收

### 验收时机
- 所有灯具、开关插座安装完毕通电后

---

### 检查清单

| 检查项 | 标准要求 |
|--------|---------|
| 灯具点亮 | 所有灯都能正常亮 |
| 开关对应 | 一个开关控制一组灯 |
| 双控功能 | 床头/门口都能关灯 |
| 亮度均匀 | 无暗区、无频闪 |
| 调光功能 | 如有调光开关需测试 |
| 智能联动 | WiFi/语音控制正常 |

---

### 插座相位检测

```bash
【工具】
相位检测仪（几十元一个）

【检测方法】
插入插座，看指示灯显示

【正常状态】
✓ 火零地接线正确
✓ 左零右火上接地
✓ 无缺相、反相、漏电

【异常提示】
✗ 缺相 → 某根线没接通
✗ 反相 → 火零接反了
✗ 漏电 → 地线问题
```

## ⚠️ 常见问题整改标准

### 常见质量问题及处理

| 问题 | 严重程度 | 整改方式 |
|------|---------|---------|
| 水电打压掉压 | 严重 | 重新找漏点、重做 |
| 防水闭水失败 | 严重 | 铲除重做、加强节点 |
| 瓷砖大面积空鼓 | 严重 | 铲掉重贴 |
| 墙面开裂贯穿 | 严重 | 切开重做 V 缝 |
| 门窗五金损坏 | 中等 | 更换配件 |
| 地板局部异响 | 中等 | 加固固定 |
| 涂料少量瑕疵 | 轻微 | 补漆 |
| 缝隙不均匀 | 轻微 | 打胶修饰 |
