## 🎯 电器选购的核心要素

```markdown
优先级排序：
1️⃣ 能效等级（决定长期使用成本）
2️⃣ 核心参数匹配（匹数/容量/尺寸等）

```

## ❄️ 空调选购指南

### 匹数选择公式

```markdown
【基本计算公式】
所需制冷量 = 房间面积（㎡）× 200W/㎡

【常见匹数对照表】
├─ 1 匹 → 适用 10-15㎡
├─ 1.5 匹 → 适用 15-22㎡（卧室主流）
├─ 2 匹 → 适用 22-30㎡
├─ 3 匹 → 适用 30-50㎡（客厅柜机）
└─ 5 匹 → 适用 50-80㎡（商用）

【特殊情况调整】
↑ 顶层/西晒/落地窗 → +0.5-1 匹
↑ 南方高温地区 → +0.5 匹
↓ 北方/有保温层 → -0.5 匹

💡 举例：
20㎡南向卧室（广州）→ 1.5 匹足够
30㎡客厅（西晒 + 顶楼）→ 建议 2.5-3 匹
```

### ❌ 避坑要点

```markdown
□ 别贪便宜买三级能效（长期电费亏死）
□ 别信"负离子净化空气"等营销概念
□ 安装费要问清楚（高空费、加长铜管费）
□ 外机位提前测量（有些户型放不下大外机）
□ 质保期看清楚（整机一般 6 年，压缩机 10 年）
```

## 🧊 冰箱选购指南

### 容量选择公式

```markdown
【基础计算】
人均容量 = 100-150L
总容量 = 人数 × 100-150L

【常见配置】
├─ 2 人小家庭 → 200-300L
├─ 3 口之家 → 300-450L
├─ 4 人以上 / 囤货族 → 450L+

【特殊情况】
↑ 爱囤生鲜 → +50-100L
↑ 经常请客 → +100-150L
↓ 单身/情侣 → 可压缩到 150L
```

---

### 循环系统类型对比

| 类型 | 原理 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|-------|
| **单循环** | 一个蒸发器 | 便宜 | 串味严重 | ❌❌ 淘汰 |
| **双循环** | 两个蒸发器 | 不串味、温控精准 | 价格高 | ✅✅✅ 首选 |
| **三循环** | 三个独立回路 | 极致保鲜 | 贵 | ⭐⭐⭐ 土豪选 |

```markdown
⚠️ 注意：很多冰箱标称"双系统"其实是假的双循环

真双循环特征：
├─ 冷藏室和冷冻室各自独立蒸发
├─ 价格一般在 4000 元以上
└─ 查看官网技术参数确认
```

### 制冷方式分类

| 类型 | 优点 | 缺点 | 推荐度 |

### 压缩机与能效

```markdown
【压缩机类型】
├─ 定频：便宜但耗电、温差大
└─ 变频：价格高、省电、温度恒定（强烈推荐）

【能效等级】
├─ 一级能效：最省电，但价格高
├─ 二级能效：性价比之选
└─ 三级能效：不推荐（耗电明显）

【日耗电量参考】
├─ 一级能效：0.4-0.6 度/天
├─ 二级能效：0.6-0.8 度/天
└─ 三级能效：0.8-1.2 度/天
```

## 🌊 净水器 & 软水机选购

### 反渗透净水器 RO vs 超滤 UF

| 类型 | 过滤精度 | 过滤效果 | 是否去水垢 | 出水口感 | 价格 |
|------|---------|---------|----------|---------|------|
| **RO 反渗透** | 0.0001μm | 除菌 + 除重金属 + 水垢 | ✅ 纯水 | ✅ 甜 | ¥¥¥ |
| **超滤 UF** | 0.01μm | 除菌但不去重金属 | ❌ 保留矿物质 | ⭐⭐⭐ | ¥¥ |

```markdown
【推荐建议】
├─ 水质差的地区 → RO 反渗透（必选）
├─ 水质好的地区 → 超滤可用（保留矿物质）
└─ 有宝宝冲奶 → RO 反渗透（更安全）
```

---

### 净水器关键参数

```markdown
【通量】（出水量）
├─ 400G → 2L/min（够用）
├─ 600G → 2.5L/min（主流）
├─ 800G → 3L/min（快速）
└─ 1000G+ → 4L/min（豪宅用）

【滤芯寿命】
├─ PP 棉：3-6 个月（前置预处理）
├─ 活性炭：6-12 个月（吸附异味）
├─ RO 膜：2-3 年（核心部件）
└─ CTO：6-12 个月（后段改善口感）

【换芯成本】（重要！）
├─ 首年免费滤芯赠送
├─ 后续年费：300-800 元/年
└─ 买之前算总账（机器 + 滤芯）

【废水比】
├─ 普通机型：2:1（2 杯净水排 1 杯废水）
└─ 高效机型：3:1 或更高
```

---

### 软水机必要性分析

```markdown
【软水功能】
├─ 去除钙镁离子 → 软化水质
├─ 保护热水器、龙头（无水垢）
├─ 皮肤洗澡更柔滑
└─ 衣服洗涤更蓬松

【适用情况】
✓ 水质硬度高（水垢严重的城市）
✓ 预算充足（软水机 + 耗材贵）
✓ 浴室面积大（考虑全屋安装）

❌ 不建议：
✗ 预算紧张（RO 净水器更重要）
✗ 水质本身就不硬（如华南部分地区）

【价格参考】
├─ 小型软水机：5000-10000 元
├─ 大型全屋软水：10000-30000 元
└─ 耗材：树脂再生盐 100-200 元/袋
```
