## 🎯 选购核心原则

```markdown
1️⃣ 先定尺寸再选型号（嵌入最易踩坑）
2️⃣ 功能需求匹配实际使用频率
3️⃣ 能效等级影响长期电费成本
4️⃣ 品牌售后比参数更重要
```

## 🍳 一、油烟机 & 灶具详细指南

### 1. 油烟机类型深度对比

| 类型 | 吸烟效果 | 适用场景 | 价格区间 | 推荐度 |
|------|---------|---------|---------|-------|
| **侧吸式** | ⭐⭐⭐⭐⭐ | 中式爆炒家庭 | 2000-6000 元 | ✅✅✅ 首选 |
| **顶吸式** | ⭐⭐⭐⭐ | 开放式厨房 | 2500-8000 元 | ✅✅ 颜值优先 |
| **升降式** | ⭐⭐⭐⭐⭐ | 追求科技感 | 4000-10000 元 | ⭐⭐ 预算充足 |
| **集成灶** | ⭐⭐⭐⭐⭐+ | 小户型/封闭厨房 | 5000-15000 元 | ✅ 省空间 |

---

#### A. 侧吸式油烟机详解

```markdown
【工作原理】
近距离吸烟（离锅 35-40cm），油烟刚产生就被吸走

【核心参数】
├─ 风量：≥20m³/min（爆炒家庭建议 22m³/min+）
├─ 风压：≥400Pa（高层住宅必须≥500Pa）
├─ 噪音：<65dB（静音型可<60dB）
└─ 能效等级：一级能效省电

【优势】
✓ 不碰头（适合高个子）
✓ 视觉上不挡视线
✓ 吸烟路径短效率高

❌ 劣势
✗ 占用灶台上方空间
✗ 滤网清洗相对麻烦
✗ 部分型号容易溅到油
```

---

#### B. 顶吸式油烟机详解

```markdown
【工作原理】
吸风口在头顶上方（T7 字型或欧式方型）

【子类型】
├─ T 型顶吸（传统款）→ 便宜实用
├─ 欧式塔式（圆润造型）→ 美观高档
└─ 超薄近吸（新款）→ 平衡两者优点

【核心参数】
同侧吸式，但安装高度更高（65-75cm）

【优势】
✓ 外观大气上档次
✓ 覆盖范围广
✓ 不占灶台上部空间

❌ 劣势
✗ 碰头风险（尤其高个子）
✗ 烟道距离长吸力有损耗
✗ 视野可能被遮挡
```

---

#### C. 集成灶详解

```markdown
【结构特点】
油烟机 + 灶具 + 消毒柜/蒸箱三合一

【吸烟原理】
下方吸烟，负压区靠近锅口（理论上最好）

【核心参数】
├─ 风量：18-22m³/min
├─ 风压：350-600Pa
├─ 热效率：≥63%（一级能效）
└─ 消毒方式：紫外线 + 高温

【优势】
✓ 吸烟效果最佳（实测数据）
✓ 节省空间（不用吊柜）
✓ 集多种功能于一体

❌ 劣势
✗ 价格偏贵
✗ 维修麻烦（一体化设计）
✗ 火力不如独立灶具大
✗ 不适合开放式厨房（可能窜味）
```

### 2. 油烟机购买决策树

```markdown
开始 → 你家人做饭习惯如何？
    │
    ├─ 天天爆炒 → 选侧吸式 ≥22m³/min
    │   └─ 高层住宅 → 确认风压 ≥500Pa
    │
    ├─ 偶尔做饭/西式烹饪 → 顶吸式足矣
    │   └─ 开放式厨房 → 考虑集成灶或带止逆阀
    │
    └─ 小户型厨房 <4㎡ → 集成灶省空间
        └─ 注意预留检修口

关键提醒：
→ 先定品牌和型号，再让橱柜商家开孔
→ 提前确认烟道口径和位置
→ 止逆阀一定要装！防止邻居倒灌
```

### 3. 燃气灶核心参数

```markdown
【热负荷】（火力大小）
├─ 小火 ≤2.0kW（煲汤不用关火）
├─ 大火 ≥4.5kW（爆炒必需）
└─ 优秀标准：≥5.0kW

【热效率】（节能程度）
├─ 一级能效 ≥63%
├─ 二级能效 ≥59%
└─ 三级能效 ≥55%

【点火方式】
✓ 脉冲电子点火（快速稳定）
✗ 压电陶瓷点火（老技术，淘汰）

【炉头材质】
├─ 不锈钢 → 性价比之王 ✅
├─ 铸铁 → 厚重耐用但易生锈
└─ 铝合金 → 廉价感强，慎选

【安全装置】（必备！）
✓ 熄火保护装置（燃气泄漏自动切断）
✓ 童锁（防误触，有小孩家庭必选）
✓ 溢锅保护（溢出时自动断电）

【燃气类型】（不能搞错！）
├─ 天然气 12T → 城市管道气
├─ 液化气 20Y → 瓶装煤气
└─ 人工煤气 → 已逐渐淘汰
```

## 🥘 二、洗碗机选购指南

### 1. 洗碗机类型对比

| 类型 | 容量 | 安装方式 | 价格 | 实用性 |
|------|------|---------|------|-------|
| **台式** | 6-8 套 | 放在台面 | 1500-3000 元 | ⭐⭐ 小空间可用 |
| **嵌入式** | 12-14 套 | 橱柜内置 | 3000-8000 元 | ✅✅✅ 主流选择 |
| **水槽式** | 4-6 套 | 替代水槽 | 6000-12000 元 | ⭐⭐⭐ 方便但贵 |

---

### 2. 碗篮容量计算

```markdown
【一套 = 一个成人一人份餐具】
包含：饭碗×1、汤碗×1、盘子×2、杯子×1、筷子×1 双

【家庭配置建议】
├─ 单身/情侣 → 6-8 套（台式/小型）
├─ 3 人家庭 → 10-12 套（中型嵌入式）
└─ 4 人以上 / 经常请客 → 13-16 套（大型）

💡 经验：宁大勿小！
买小了每次要分批次洗反而更费水费电
```

---

### 3. 洗碗机核心功能

```markdown
【必选功能】
✓ 热风烘干（比普通烘干效果好）
✓ 智能洗涤（根据脏污程度调整）
✓ 分层洗涤（少量餐具用半载模式）
✓ 延时预约（错峰用电）
✓ 除菌模式（高温/紫外线）

【加分功能】
☆ 开门助干（洗完自动开缝晾干）
☆ 水晶胆烘干（内胆不粘油）
☆ APP 远程控制（手机查看状态）
☆ 水龙头取水（无需频繁加水）

【智商税功能】
✗ 语音控制（没必要）
✗ 显示屏超大屏（浪费钱）
✗ 花哨的灯光效果（无实质作用）
```

---

### 4. 洗碗机安装注意事项

```markdown
【尺寸核对】
宽度：标准 60cm（也有 45cm/75cm 窄版）
高度：80-85cm（需确认是否可调节脚）
深度：55-60cm（含门板厚度）

【水电预留】
□ 电源插座：独立回路，带漏电保护
□ 进水口：冷水管接口（0.1-1MPa 水压）
□ 排水口：距地 50-80cm，高于机器
□ 进排水管长度：预留足够余量

【橱柜定制要点】
→ 先确定洗碗机型号再订橱柜
→ 确认是否需要专用台面
→ 侧面留出散热空间（如有）
→ 底部留空方便拉出清洁
```

## 💧 三、净水器 & 软水机进阶指南

### 1. 反渗透净水器 RO vs 超滤 UF

| 类型 | 过滤精度 | 去除物 | 保留矿物质 | 出水口感 | 适用地区 |
|------|---------|-------|----------|---------|---------|
| **RO 反渗透** | 0.0001μm | 细菌/病毒/重金属/水垢 | ❌ | ✅ 纯水甜 | 水质差地区首选 |
| **超滤 UF** | 0.01μm | 细菌/悬浮物 | ✅ | ⭐⭐⭐ | 水质好的地区 |

---

### 2. RO 净水器关键参数

```markdown
【通量】（出水量）
├─ 400G → 2L/min，够用 | 1500-2500 元
├─ 600G → 2.5L/min，主流 | 2000-3500 元
├─ 800G → 3L/min，快速 | 2500-4500 元
└─ 1000G+ → 4L/min，豪宅用 | 4000-8000 元

【滤芯寿命】
├─ PP 棉：3-6 个月（前置预处理）
├─ 活性炭：6-12 个月（吸附异味）
├─ RO 膜：2-3 年（核心部件）
└─ CTO：6-12 个月（后段改善口感）

【换芯成本】（重要！购买前算总账）
├─ 首年免费滤芯赠送
├─ 后续年费：300-800 元/年
└─ 综合计算：机器价格 + 3 年滤芯费

【废水比】
├─ 普通机型：2:1（排 1 杯废水出 2 杯净水）
├─ 高效机型：3:1 或 4:1
└─ 一级节水：≤1:1（极少见且贵）

【认证标准】
✓ NSF 认证（美国净水器权威认证）
✓ CMA 检测报告
✓ 涉水批件（国内强制）
```

### 4. 软水机必要性分析

```markdown
【软水功能】
→ 去除钙镁离子，软化水质

【适用情况】
✓ 水质硬度高（水垢严重的城市）
✓ 预算充足（软水机本身 + 耗材贵）
✓ 浴室面积大（考虑全屋安装）
✓ 有老人小孩皮肤敏感

❌ 不建议
✗ 预算紧张（RO 净水器更重要）
✗ 水质本身就不硬（华南部分地区不需要）
✗ 租房短期居住（不值得投资）

【价格参考】
├─ 小型单级软水机：5000-10000 元
├─ 大型双罐软水机：10000-30000 元
└─ 耗材：树脂再生盐 100-200 元/袋

【维护成本】
→ 每 3-6 个月加一次再生盐
→ 每 3-5 年更换一次树脂
→ 每年检查阀门密封性
```

## 🏠 四、新风系统选购

### 1. 新风系统类型对比

| 类型 | 原理 | 适用场景 | 价格 | 优缺点 |
|------|------|---------|------|-------|
| **中央新风** | 全屋管道换气 | 装修前规划 | 1-3 万元 | 效果好但需吊顶 |
| **壁挂新风机** | 单墙钻孔换气 | 已装修房屋 | 3000-8000 元 | 安装简单效果一般 |
| **管道式单机** | 单房间净化 | 局部区域 | 2000-5000 元 | 灵活但覆盖有限 |

---

### 2. 中央新风系统参数

```markdown
【风量选择】
公式：房间体积 × 换气次数 = 所需风量

举例：
100㎡房子，层高 2.8m → 体积 280m³
每小时换气 1 次 → 需要 280m³/h 风量
建议选择：350-450m³/h（留有余量）

【过滤等级】
├─ G3/G4 → 粗效过滤（颗粒物）
├─ F7/F8 → 中效过滤（PM2.5）
└─ H13/H14 → 高效 HEPA（医用级）

【热回收】
✓ CRV（显热交换）→ 温度交换
✓ ERV（全热交换）→ 温湿度同时交换

【噪音控制】
├─ 低速档：<30dB（睡眠模式）
├─ 中速档：35-40dB（日常使用）
└─ 高速档：45-50dB（快速换气）

【能耗】
→ 连续运行约 30-60W/小时
→ 一天约 0.7-1.5 度电
```
