## 🔍 断桥铝窗详细解析

### 型材壁厚标准

```markdown
【新国标要求】
├─ 外窗：≥1.8mm
├─ 门：≥2.0mm
└─ 旧国标允许：≥1.4mm

【市场分级】
├─ 入门级：1.4mm（符合旧国标，够用）
├─ 主流级：1.6-1.8mm（推荐）
└─ 高端级：2.0mm+（豪宅用）

⚠️ 警惕"超薄型材"：
   └─ 有些小厂出 1.2mm 甚至更薄的，千万别买！
```

### 断桥条材质

```markdown
【PA66GF25 尼龙条】✅ 推荐
├─ 优点：耐热、耐低温、强度高
├─ 特点：灰色或黑色，表面有纹路
└─ 标识：上面印有 PA66GF25 字样

【PVC 塑料条】❌ 避坑
├─ 问题：易老化、收缩、开裂
└─ 寿命：3-5 年就开始失效

如何鉴别：
├─ 看标识：正规产品会标注材质
├─ 闻气味：PA66 无异味，PVC 有刺鼻味
└─ 掰一下：PA66 硬度高不易变形
```

---

#### 🧊 双层中空玻璃规格

```markdown
【基本结构】
5mm+A12A+5mm = 双层 5mm 钢化玻璃 + 12mm 空气层

【参数说明】
├─ 第一组数字：外层玻璃厚度
├─ A12A：中间空气层 12mm（也可充氩气）
└─ 最后一组数字：内层玻璃厚度

【常用配置】
├─ 普通住宅：5+12A+5mm（性价比高）
├─ 高层：5+16A+5mm（隔音更好）
├─ 临街：5+12A+5mm + Low-E 镀膜
└─ 超静音：6+12A+6mm 夹胶中空

【认证标志】
✓ 3C 强制认证（钢化玻璃必须有）
✓ 工厂代码、温度标记、生产日期
```

---

#### 💎 Low-E 镀膜玻璃详解

```markdown
【什么是 Low-E】
Low-Emission 低辐射镀膜 → 反射红外线，减少热量传递

【功能】
├─ 夏天：阻挡室外热辐射（降低空调能耗）
├─ 冬天：保持室内热量不散失
└─ 防晒：阻隔紫外线（保护家具褪色）

【类型】
├─ 单银 Low-E → 性价比之选
├─ 双银 Low-E → 性能更强
└─ 三银 Low-E → 顶级配置，贵

【适用情况】
✓ 西向/南向窗户（强烈建议加）
✓ 顶层房间（隔热需求强）
✓ 寒冷地区（保温优先）

❌ 不适合：
✗ 北向采光不足的房间（可能影响进光）
```

### 防盗门核心指标

```markdown
【甲级门 > 乙级 > 丙级 > 丁级】
判断方法：看门框上的钢印标识

甲级门标准：
├─ 门厚≥90mm
├─ 锁具：C 级锁芯（最安全）
├─ 合页：3 个以上内置防撬
├─ 防技术开启：≥6 分钟
└─ 防暴力开启：≥15 分钟

【锁具选择】
✅ C 级直插式锁芯（首选）
✅ 智能锁（指纹/密码/APP 远程控制）
❌ B 级以下锁芯（别考虑）

【填充物】
├─ 航空铝箔（最好，防火隔音）
├─ 岩棉（不错，性价比高）
└─ 纸蜂窝（劣质，坚决不要）
```

## 🚫 门窗选购常见陷阱

### ❌ 陷阱 1：壁厚虚标

```markdown
典型操作：
├─ 说 "1.8mm 型材"
├─ 实际只有 1.4-1.5mm
└─ 用游标卡尺一测就知道

应对：
├─ 签合同注明壁厚误差±0.1mm
├─ 进场时现场测量
└─ 不符就拒绝收货
```

### ❌ 陷阱 2：铝材牌号混淆

```markdown
不同铝材：
├─ 6063-T5：普通建筑铝材（合格）
├─ 6063-T6：更高强度（优质）
└─ 再生铝：回收废料（劣质，强度差）

商家话术：
"我们的铝材特别好" → 但不说具体牌号

应对：
└─ 要求写明 6063-T5/T6
```

### ❌ 陷阱 3：五金件偷换

```markdown
套路：
├─ 报价时用进口五金品牌
├─ 安装时换成杂牌或国产贴牌
└─ 价格差三倍

应对：
├─ 合同中列明每个五金的品牌型号
├─ 收货时核对实物标签
└─ 拍照留证
```

### ❌ 陷阱 4：面积计算猫腻

```markdown
常见计费方式：
├─ 按洞口面积计算（简单粗暴）
├─ 按展开面积计算（容易多算）
└─ 按扇数计算（适合小窗户）

陷阱：
├─ 把边框、中挺等额外收费
├─ 转角立柱单独计价（一个几百块）
└─ 开启扇数量加价（每扇 +200-500 元）

应对：
└─ 提前问清楚所有收费项目
```

### ❌ 陷阱 5：玻璃以次充好

```markdown
常见问题：
├─ 非钢化玻璃冒充钢化玻璃
├─ 3C 标志模糊或缺失
├─ 中空层充空气当充氩气
└─ 双层变单层

应对：
├─ 检查玻璃上的 3C 标志是否清晰
├─ 敲击听声音（钢化玻璃声音清脆）
└─ 看截面层数
```

## 💡 常见问题速查

### Q1: 要不要做内倒窗？

```markdown
【推荐情况】
✓ 南方梅雨季节（通风不淋雨）
✓ 有老人小孩的家庭（微通风更安全）
✓ 临街环境（噪音大时可微开换气）

【不推荐情况】
✗ 老小区（防盗风险）
✗ 底层住户（容易被爬入）
✗ 小户型（开启占空间）

结论：
→ 主卧、次卧推荐配内倒功能
→ 客厅和厨房可以不配
```

### Q2: 临街噪音大怎么办？

```markdown
【最强降噪方案】
1. 三层中空玻璃（6+12A+6）
2. 夹胶中空玻璃（5+0.76PVB+5+12A+5）
3. 系统窗（整体密封 better）
4. 加装新风系统（不开窗也能换气）

【性价比方案】
1. 双层中空玻璃 5+16A+5
2. 加 Low-E 镀膜（还隔热）
3. 断桥铝 70 系列以上
4. 三元乙丙密封胶条

【关键细节】
└─ 多道密封：至少 3 道（玻片密封 + 窗扇密封 + 框扇密封）
```

### Q3: 西晒房间窗户怎么选？

```markdown
【重点需求】
├─ 遮阳隔热（降低空调负荷）
├─ 防晒（保护家具不褪色）
└─ 透光率不能太低

【推荐方案】
├─ Low-E 镀膜玻璃（强烈推荐）
├─ 双层中空 + 充氩气（保温更好）
├─ 外遮阳卷帘（物理遮阴）
└─ 内侧窗帘布（双重防护）

【不推荐】
└─ 深色玻璃（影响采光）
```

### Q4: 封阳台安全吗？

```markdown
【必须做到的安全措施】
✓ 玻璃厚度：≥6mm（大于 2㎡用 8-10mm）
✓ 型材壁厚：≥1.8mm（断桥铝）
✓ 固定方式：化学螺栓固定
✓ 护栏高度：≥1.1m（儿童安全）
✓ 限位器：防止窗户开启过大

【物业规定】
└─ 提前了解是否允许封阳台
```
