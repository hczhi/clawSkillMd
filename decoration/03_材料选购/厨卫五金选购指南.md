#### 托底隐藏轨详解

```markdown
【结构特点】
├─ 安装在抽屉底部，从外面看不到
├─ 三节式导轨（完全拉出）
├─ 内置阻尼缓冲

【优点】
✓ 美观（抽屉拉开也看不到轨道）
✓ 承重强（可放重物如餐具）
✓ 顺滑静音（带阻尼缓冲）
✓ 易清洁（不易积灰）

【缺点】
✗ 价格贵（比普通轨贵 2-3 倍）
✗ 安装要求高（需专业师傅）

【品牌推荐】
├─ Blum Tip-On（带弹出来功能）
├─ Hettich InnoTech
├─ DTC 托底阻尼轨（高性价比）
```

#### 💡 抽屉配置方案

```markdown
【厨房操作台下方】

└─ 转角处 → 飞碟拉篮 / 小怪物拉篮

【衣柜内部】
└─ 每个吊柜下方配 2-3 个浅抽屉（内衣袜子）
```

#### ✅ 花洒参数标准

```markdown
【基本要求】
├─ 进水口标准：G1/2（4 分接口）
```

#### ✅ 地漏选型原则

```markdown

├─ 必须：密封性好、防溢水
└─ 尺寸：对应洗衣机排水管直径

【频率使用少的房间】→ T 型弹簧式
├─ 原因：不会干涸
└─ 适用：客卫、阳台
```

## 🚫 五金选购常见陷阱

### ❌ 陷阱 1："进口五金"纯贴牌

```markdown
套路：
├─ 宣称德国/奥地利进口
├─ 实际是国内小厂代工
└─ 价格虚高三倍

应对：
├─ 索要原产地证明
├─ 查官方网站是否有此型号
└─ 看铭牌上的详细生产信息
```

### ❌ 陷阱 2：材质以次充好

```markdown
典型偷工：
├─ 说"全铜"实为锌合金
├─ 说"陶瓷阀芯"实为橡胶
├─ 说"304 不锈钢"实为 201
└─ 说"PVD 镀层"实为普通电镀

应对：
├─ 验货时称重（铜的最重）
├─ 要求出具材质检测报告
├─ 看表面标识是否清晰
```

### ❌ 陷阱 3：假阻尼真卡顿

```markdown
操作手法：
├─ 新买的铰链有明显阻尼
├─ 用半年后阻尼失效
└─ 关门砰的一声巨响

应对：
└─ 选知名品牌的阻尼铰链（百隆/海蒂诗/DTC）
```
