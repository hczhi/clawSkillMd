### 第三条 增项控制
```
1. 除甲方书面确认的设计变更外，增项总额不得超过合同总价的 5%（¥1,439）。

4. 所有增项须经甲方书面（微信/短信文字）确认后实施，否则甲方有权拒绝支付。
```

### 第四条 质保条款
```
基础装饰工程：保修 2 年（从竣工验收之日起）
防水工程：保修 5 年（卫生间、厨房、阳台）
水电隐蔽工程：保修 10 年（含电路、水路）

保修响应：接到报修后 24 小时内响应，48 小时内上门
保修范围：非人为损坏的质量问题全部免费维修
```

### 第五条 材料验收
```
1. 进场前须提前 3 天通知甲方验收材料品牌和型号是否与合同约定一致。
2. 如发现有假劣质产品，甲方有权要求双倍赔偿并解除合同。
3. 关键材料品牌确认：
   - 电线：**南粤牌 BV/RV 阻燃系列**（CCC 认证）
   - 线管：**联塑 PVC 电线管**
   - 水管：**日丰 PPR Φ25 S2.5+Φ20 S3.2**（抗菌管）
   - 排水管：**联塑 PVC**
   - 防水涂料：**德高 K11 系列**（柔韧型Ⅱ型地面 + 刚性 I 型墙面）
   - 瓷砖胶：**德高 C1E 级**
   - 铝扣板：**0.8mm 国标**
```

### 第六条 工艺标准
```
1. 墙压地工艺（卫生间、厨房必须执行）
2. 海棠角阳角处理（美观耐用）
3. 闭水试验至少 48 小时，楼下邻居家派人来看并签字确认
4. 水电打压测试 0.8MPa 保压 30 分钟，压降≤0.05MPa
5. 拍照留底所有管线走向图（水印照片，带时间地点）
6. 瓷砖空鼓率<15%，平整度误差≤3mm/2m
7. 水管铺设左热右冷、上热下冷、间距 15cm
```

### 第七条 安全责任
```
1. 施工期间乙方对施工人员购买意外伤害保险。
2. 施工期间发生的一切安全事故由乙方全权负责。
3. 乙方不负责消防管道拆改、天然气管道拆改、承重墙拆除。
4. 装修期间水费电费由甲方承担。
```

### 第八条 争议解决
```
如双方对工程质量产生争议，可共同委托广州市建筑装饰行业协会进行检测，检测结果为最终依据，检测费用由责任方承担。
```
