## 💰 七、仓位建议计算

### 基于 Kelly 公式的仓位优化

**参数设定**:
- Win Rate (胜率): 70% (历史数据支持此概率)
- Win/Loss Ratio (盈亏比): 2.5 (上行空间~30%, 下行风险~-15%)

**Kelly 最优仓位**:
```
f* = (p × b - q) / b
   = (0.70 × 2.5 - 0.30) / 2.5
   = (1.75 - 0.30) / 2.5
   = 0.58 或 58%
```

**半凯利 (保守做法)**: f*/2 = **29%**

### 结合投资组合约束调整

| 考虑因素 | 调整幅度 | 原因 |
|----------|----------|------|
| 单一个股上限 | -15% | 避免集中度过高 |
| 行业集中度限制 | -5% | 消费/传媒板块已有配置 |
| 流动性考量 | -3% | A 股大市值，建仓周期长 |
| **调整后仓位建议** | **6% ~ 10%** | 单只个股适中配置 |
